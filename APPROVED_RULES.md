# 全20課で固定する暫定ルール

全20課の人間確認後に「承認済み」へ変更する。現段階では、横断監査と修正のための暫定基準。

1. 全ファイルを同じ階層に置く。
2. UI、再生処理、教材データを分離する。
3. 主要項目は「タミル文字→表面形（必要時）→構造ローマ字→カタカナ→日本語」の順で表示する。
4. タミル文字の行全体をタップすると、そのタミル語だけを再生する。
5. 例文だけでなく、短い句、代名詞、格変化した完成語にも個別音声を付ける。
6. 速度・繰返し設定は各カードへ置かず、下部の共通操作へ集約する。
7. 語形変化の途中段階は、重要例だけ「変化を見る」で展開する。
8. 接辞単独より、単語→複数形→格付き完成形の順で聞かせる。
9. 長い語は最後から分解し、「名詞―複数―格」の順序を繰り返し確認する。
10. 文法説明は「何を表すか→どこに付くか→どんな形か→なぜ必要か→似た形との差→後の課への接続」で書く。
11. 説明は規則表だけで終わらせず、映画や旅行で何を聞き取れるかへ接続する。
12. 文化補足は文法本文と視覚的に区別し、ステレオタイプな観光説明ではなく、レジスター、人間関係、字幕と台詞、借用語、移動場面など言語理解に直結する内容にする。
13. 660〜1099pxでは長い例文カードを一列に保ち、タブレットでの過密を避ける。
14. 二列化は、短い項目または1100px以上で十分な横幅がある場合に限る。
15. デザインは青緑、マンゴー、ブーゲンビリア、アイボリーを軸にするが、強い色は一画面で競合させない。
16. 南国感は、写真や派手な柄ではなく、海、日差し、葉、コーラムを抽象化した線と余白で作る。
17. 口語形と保守的綴りを無表示で混在させない。採用レジスターは `LANGUAGE_CHECK.md` に記録する。
18. 不確実な語形・口語度・文化的含意は、アプリ本文で断定せず、ネイティブ確認項目へ回す。
19. 課は原則2課ずつ追加し、制作→構文検査→実機確認→言語確認→暫定固定の順で進める。
20. 同じ形が複数の関係を表す場合、訳語一覧ではなく、後続する名詞・動詞と文全体から判定させる。
21. 場所と移動は、位置・到達点・起点の三項を同じ名詞で対比して教える。
22. 人の所在・手元を表す `iṭam` と、物・場所の `-il` をまず対照させ、例外や拡張は後続課へ送る。
23. 映画で頻出する縮約形は文化カードで示せるが、主要パラダイムへ入れる前に地域差と中立性を確認する。
24. 新しい課の追加だけで表現できる限り、共通UIを変更しない。UI変更は複数課で再利用できる場合に限る。
25. 後置詞は単独の訳語ではなく、「前の名詞がどの格形になるか」と組で教える。
26. 接合子音は意味を持つ接辞として数えず、完成形・構造形・発音の三つを混同しない。
27. 会話で格標識が表面に出ない形を扱う場合、まず構造が透明な完全形を提示し、その後で口語形を対比する。
28. 現在の名詞文、存在文、所有文、与格経験者文を、日本語の「です・ある・持つ」で一括処理しない。
29. `இருக்கிறது` は存在・所在、ゼロコピュラは現在の同一・分類、`இல்லை` は不在・非所有・名詞文否定として区別する。
30. `என்னிடம்` と `எனக்கு` は、手元・領域と経験者・受け手の代表的対照として教えるが、実際の分布が完全な二分ではないことを内部メモへ残す。
31. 第7課以降で動詞を示すときは、基本を「動詞の核―時制を作る部分―人称・数・性の末尾」の順で表示する。
32. 人称語尾は時制と切り離して比較し、`-ēṉ` と `-ōm` が現在・過去・未来に共通して現れることを繰り返す。
33. 動詞クラスは番号一覧の先行暗記ではなく、頻出動詞を現在・過去・未来の三点セットで比較して導入する。
34. 表面形と形態境界が大きくずれる過去幹は、存在しない語幹や接辞を捏造せず、完成した過去幹として表示できる。
35. `பார்க்கிறேன்` 型の保守的な主要表示と、`பாக்கறேன் / பாக்குறேன்` 型の映画口語を視覚的に分離する。
36. 単純現在を日本語の「〜している」だけに固定せず、進行中・習慣・文脈依存の現在として説明する。
37. 一人称未来は予測だけでなく、意志・予定・約束を表すことを例文で示す。
38. 一人称複数未来が勧誘として読まれる場合は、その用法を第15課へ接続し、未来の基本説明と混同しない。
39. 自動詞・他動詞は日本語訳ではなく、動作が直接向かう目的語を取るかという文内関係から説明する。
40. 親しい単数への命令形は動詞の核を見る入口として使えるが、辞書形・不定詞・丁寧依頼と同一視しない。
41. 否定は肯定三時制の単純な反転として教えず、`-வில்லை`・`மாட்டு`・`முடியாது` が否定する意味領域を分ける。
42. `-வில்லை` 型は人称語尾を持たないことを明示し、人物は主語代名詞・文脈から回収させる。
43. `-வில்லை` の過去・未完了・現在不成立の読みは、`நேற்று / இன்னும் / இப்போது` などの時間語と組で教える。
44. `மாட்டேன் / மாட்டோம்` は未来否定だけでなく、習慣的非実行・拒否・意志的な非実現を表し得ることを示す。
45. 能力・状況の不可能は `முடியாது` で扱い、意志的な拒否を表す `மாட்டேன்` と混同しない。
46. 映画口語の `பாக்கல / போகல` は、主要表示 `பார்க்கவில்லை / போகவில்லை` と視覚的に分離する。
47. 不定詞は「動詞を次の述語へ渡す非定形」として教え、時制・人称を完成させる有限動詞と区別する。
48. 不定詞形成は一律の接辞追加として単純化せず、`பார்→பார்க்க`、`வா→வர`、`செய்→செய்ய` の完成ペアで示す。
49. 目的・可能・必要・否定への接続では、同じ不定詞の後ろに何が続くかを比較させる。
50. 不定詞を含む長い文では、最後の有限動詞・述語を先に探し、左側の不定詞との意味関係を組み立てる。
51. `வரச் சொல்லுங்கள்` などの接合子音は、引用・命令の詳説を先取りせず、完成形と後続課への入口として扱う。
52. 副動詞列では、最後の有限動詞だけが時制・人称を完成させ、前の動詞は順番・方法・様子として接続すると教える。
53. 副動詞形は一律の語尾追加として扱わず、`வா→வந்து`、`போ→போய்`、`செய்→செய்து`、`வாங்கு→வாங்கி` の完成ペアで示す。
54. 基礎的な副動詞列では通常同じ主語を共有することを示すが、絶対的な統語規則として過度に一般化しない。
55. `-விட்டு` は前の動作の完了・一区切り、`-கொண்டே` は継続・同時、`-ஆமல்` は前の動作を行わないこととして対比する。
56. 否定の位置を比較し、`பார்க்காமல் போனேன்` と `பார்க்க போகவில்லை` の意味の違いを形のまとまりから説明する。
57. 補助動詞列は、右端の時制・人称から左へ戻り、進行・完了・保持・試行を一層ずつ復元させる。
58. `பார்த்துக்கொண்டிருக்கிறேன்` は単純現在より進行を明示する形として示すが、両者の意味領域が重なることも説明する。
59. `முடி` は終点到達、`விடு` は完了・既済・態度、`வை` は結果保持・準備、`பார்` は試行として代表例を示す。
60. 補助動詞の日本語訳を一語に固定せず、元の動詞の意味と場面から文法的な意味が伸びることを説明する。
61. 映画口語の `பாத்துக்கிட்டு இருக்கேன் / பாத்துட்டேன் / வாங்கி வெச்சேன்` は、主要表示から分離し、基本形へ戻す聞き取り練習として扱う。
62. 可能・希望・必要・禁止・意志は、日本語の助動詞置換表ではなく、不定詞の後ろに何が続くかで比較する。
63. `முடியும் / முடியாது` は能力・条件、`கூடாது` は規則・許可の禁止として対比する。
64. `வேண்டும்` は希望・必要・義務へ広がるため、時間・人物関係・場面から強さを判断させる。
65. 個人的な願いを明示するときは `எனக்கு ... ஆசை இருக்கிறது` を用い、`வேண்டும்` との意味差を示す。
66. `-ப் போகிறேன்` は移動目的だけでなく、具体化した予定・近い未来・目前の予測へ広がることを説明する。
67. 全人称は巨大表を先に見せず、「一・二人称」「人間三人称」「非人間」の三群に分けて導入する。
68. 人称語尾は同じ人物について現在・過去・未来を横断して比較し、時制幹と人称末尾を分離する。
69. `அவர் ... -ஆர்` は性別に依存しない敬意形として教え、`அவர்கள் ... -ஆர்கள்` の人間複数と区別する。
70. 非人間活用は人間活用の単純な語尾交換とせず、単数 `-atu / -um`、複数 `-aṉa`、未来の `-um` 共用を独立して示す。
71. 映画口語の `பாக்கற / பாக்கறீங்க / பாக்கறான் / பாக்கறா / பாக்கறாங்க / வருது` は、人物関係を読む資料として扱い、完全形と混在させない。
72. 命令・依頼は語形の強弱だけでなく、親しい単数・丁寧単数・複数という相手との距離として教える。
73. 丁寧形 `-உங்கள்` と口語の `-ுங்க` を分離し、`வாருங்கள் / வாங்க` のような縮約・多義性は文化カードへ送る。
74. 否定命令 `-ஆதே / -ஆதீர்கள்` は、話し手自身の未来否定 `மாட்டேன்` と主語・発話機能を対比する。
75. `-லாம் / -லாமா` は勧誘・提案・許可へ広がるため、包含一人称や明示された主語から読みを決める。
76. はい・いいえ疑問は基本語順を保ち、述語へ付く `-ஆ` を目印にする。疑問詞疑問では語順反転を前提にしない。
77. 名詞句は指示・所有・数・性質を名詞の前へ置き、複数・格語尾は中心名詞の後ろへ付ける。
78. 形容詞は中心名詞の性・数・格に一致して変化しない。基本形容詞と `-ஆன` 派生形を分けて示す。
79. 数詞後の複数標識はレジスター・語彙で揺れる可能性を明示し、一つの例から絶対規則へ一般化しない。
80. 比較は基準名詞＋`விட`、最上級的表現は集合＋`மிகவும்` などで作り、専用の一語尾として教えない。
81. 連体修飾は関係代名詞の置換表ではなく、連体形＋後続する中心名詞として教える。
82. 連体形は人称語尾を持たないことを明示し、行為者は節内の主語または文脈から復元させる。
83. 同じ動詞で `-கிற / 過去連体形 / -உம் / -ஆத` を比較し、現在・過去・未来／習慣・否定を横断して示す。
84. `-உம்` 連体形を未来専用とせず、予定・習慣・一般的性質を名詞へつなぐ形として説明する。
85. 中心名詞が連体節内で主語・目的語・場所・相手などのどの役割を持つかは、動詞と文脈から判断させる。
86. 連体節を含む長い名詞句でも、複数・格語尾は最後の中心名詞へ付ける。
87. `நான் பார்த்த படம்` と `நான் பார்த்தது` を対比し、中心名詞ありの連体修飾と、中心名詞なしの名詞化を分ける。
88. 名詞化は一つの接辞を機械的に付ける規則として教えず、`பார்ப்பது / பார்த்தது / பார்க்காதது` 等の完成組で示す。
89. 名詞化された節も格を取り、`-தை / -தற்கு` 等の末尾から文全体での役割を確認させる。
90. 名詞化形が「もの」を指すか「出来事・事実」を指すかは、節内の目的語と前後の文脈から判断させる。
91. `என்பது` は完成した文を「〜ということ・事実」として包む形とし、次課の `என்று` に接続する。
92. 書記的な `-தல்` 名詞化は口語中心の主要パラダイムへ混ぜず、短い補足に留める。
93. 映画口語の `பாக்கற படம் / வர்ற ஆள் / பாக்கறது / பாத்தது` は、連体形か名詞化かを後続名詞と語末から見分ける練習として扱う。
94. `என்று` は前の文・語・音を一つの引用内容としてまとめ、後ろの発言・思考・質問動詞へ渡す境界として教える。
95. 直接引用では、引用内の人称・時制・時間語が元の話者の視点を保つことを示す。
96. 疑問の引用では、疑問詞または `-ஆ` が引用箱の中で質問を作り、`என்று` 自体を疑問標識として扱わない。
97. `வா என்று சொன்னார்` と `வரச் சொன்னார்` を対比し、言葉そのものの引用と、指示された行為の報告を分ける。
98. `என்பது` は内容を名詞として扱い、`என்று` は内容を後ろの発言・思考動詞へ渡すという入口の対比を保つ。
99. 映画口語の `-ன்னு / -னு` は `என்று` に対応する引用標識として扱い、主要形と混在させない。
100. 複文は最後の有限動詞・述語を先に確認し、左側を節末の標識から条件・理由・目的・時間へ分ける。
101. 条件の `-ஆல்` は過去専用とせず、まだ実現していない条件・一般条件にも使う形として示す。
102. 否定条件は `இல்லையென்றால் / வரவில்லை என்றால்` の透明な形から入り、`வராவிட்டால்` と口語短縮へ広げる。
103. `-ஆலும்` は条件を認めても後ろの結果が変わらない譲歩として、単純条件 `-ஆல்` と対比する。
104. 理由の `-தால்` は名詞化された出来事＋道具格に由来する形として示すが、歴史的分析より現在の完成形を優先する。
105. 目的は短い不定詞と `-தற்காக` の二形を示し、日常的な簡潔さと目的の明示・強調の違いを説明する。
106. 時間節は `-போது / முன் / பிறகு / வரை` を節境界として見つけ、基準となる出来事との前後・同時・終点を読む。
107. 複数の従属節が重なる文でも、節を左から逐語訳せず、最後の述語と各節末標識から階層を復元する。
108. 映画口語の `வந்தா / இல்லனா / ஏன்னா / வர்றப்போ / வந்தப்புறம்` は文化カードで示し、地域差を確認する。
109. 全20課完成後は新機能を先に増やさず、実機・母語話者・語彙・ローマ字・カタカナ・TTSの横断監査を優先する。
