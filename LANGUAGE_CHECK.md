# 言語検証メモ｜第1〜16課 増補版

画面へ表示しない内部確認用メモ。第17課以降の制作前に、母語話者またはタミル語学の専門家による確認を行う。

## 1. 教材レジスター

- 中心は、チェンナイを含むタミル・ナードゥ州で広く理解される口語への入口。
- 主要例文は形態構造とTamil TTSの安定性を優先し、比較的保守的な綴りを残す。
- 映画会話で頻出する `பாக்கறேன் / போறேன் / பாக்கல / பாத்துட்டேன் / வெச்சேன் / வேணும் / பாருங்க` 等は文化カードへ分離する。
- 代名詞では `நாங்க / நாம / நீங்க / அவங்க`、所有では `-ஓட` を明示的な口語形として採用する。
- 第14課以降も、主要活用表は構造が見える形、文化欄は会話音へ戻す形、という二層方式を維持する。

## 2. 第1〜14課から継続する確認

### 名詞・格・後置詞

- `இதுக்கு / அதுக்கு` と書記形 `இதற்கு / அதற்கு`。
- `என் / என்னோட`、`உன் / உன்னோட` の語用差。
- `-ஓட` の所有・共同用法と `சேர்ந்து` の併用。
- `வீட்டில் / என்னிடம்` と口語 `வீட்டுல / என்கிட்ட` の距離。
- `படத்தைப் பற்றி / படம் பற்றி` の対格表示。
- `-க்காக / பதிலாக / தவிர` の格支配と接合子音。

### 名詞文・存在・経験者

- `இது என் வீடு` のゼロコピュラ説明。
- `இது என் வீடு இல்லை` と `வீடல்ல` 系の自然さ・分布。
- `என்னிடம் ... இருக்கிறது` と `எனக்கு ... இருக்கிறது` の所有表現上の重なり。
- `எனக்கு ... பிடிக்கும் / வேண்டும் / புரியும்` の与格経験者。

### 動詞・否定・非定形

- `பார்க்கிறேன் / பாக்கறேன் / பாக்குறேன்` の地域差。
- `பார்த்தேன்` を `pār-tt-ēṉ`、`போனேன்` を `pō-ṉ-ēṉ` と示す判断。
- `சாப்பிட்டேன்` を `cāppiṭṭ-ēṉ` とし、存在しない形態素を作らない方針。
- 単純現在による進行中・習慣の読み。
- `-வில்லை` の過去・未完了・現在不成立の読み分け。
- `மாட்டேன்` の未来否定・習慣・拒否の範囲。
- `என்னால் / எனக்கு ... முடியாது` の分布。
- `பார்→பார்க்க / வா→வர / செய்→செய்ய / வாங்கு→வாங்க` の不定詞表示。
- `வந்து / போய் / செய்து / பார்த்து / வாங்கி / சாப்பிட்டு` の副動詞表示。
- `-விட்டு / -கொண்டே / -ஆமல்` の意味範囲。
- `பார்த்துக்கொண்டிருக்கிறேன்` と単純現在の対比。
- `முடி / விடு / வை / பார்` の補助動詞用法。

### モダリティ・全人称

- `முடியும் / முடியாது` と能力者 `என்னால்` の分布。
- `ஆசை இருக்கிறது / வேண்டும் / விரும்புகிறேன்` の希望表現の距離。
- `வேண்டும் / -வே வேண்டும் / கூடாது` の強さ。
- `-ப் போகிறேன்` の移動目的・近接未来・意志・予測。
- `-āy / -īrkaḷ / -āṉ / -āḷ / -ār / -ārkaḷ` の三時制対応。
- `அவர் / அவர்கள்` と口語 `அவங்க` の敬意・複数。
- 非人間単数 `வருகிறது / வந்தது / வரும்`。
- 非人間複数 `வருகின்றன / வந்தன / வரும்` と口語での一致平準化。

## 3. 第15課：命令・依頼・勧誘・疑問

### 親しい命令と丁寧形

主要対比：

- `வா → வாருங்கள்`
- `போ → போங்கள்`
- `பார் → பாருங்கள்`
- `கேள் → கேளுங்கள்`
- `சொல் → சொல்லுங்கள்`

教材では前者を親しい一人、後者を丁寧な単数・複数として説明する。

要確認：

- `போங்கள்` と、より実際的な口語 `போங்க` の距離。
- `வாருங்கள்` と口語 `வாங்க` の対応。`வாங்க` は「買ってください／買う」の形とも同形になり得るため、文脈説明が必要。
- `கேள்` が「聞く／尋ねる」の双方を担う範囲。
- `ஒரு நிமிடம் காத்திருங்கள்` の中立性。

### 依頼の軟化

- `கொஞ்சம் ...` を「少し」の量だけでなく、依頼を和らげる語として扱う。
- `கொஞ்சம் உதவி செய்ய முடியுமா?`、`இந்த முகவரியைச் சொல்ல முடியுமா?` を能力疑問から丁寧依頼へ伸びる例とする。

要確認：

- `கொஞ்சம் உதவி செய்ய முடியுமா?` の自然さ。
- `முடியுமா?` が年長者・初対面へ十分丁寧か。
- `தயவு செய்து` を基礎本文へ入れない方針の妥当性。

### 否定命令

- 親しい単数：`-ஆதே -ātē`
- 丁寧・複数：`-ஆதீர்கள் -ātīrkaḷ`
- `போகாதே / போகாதீர்கள்`
- `பேசாதே / போனை பயன்படுத்தாதீர்கள்`

`போக மாட்டேன்` は「私は行かない」、`போகாதே` は「あなたは行かないで」と説明する。

要確認：

- 語幹分析 `pōk-ātē / pēc-ātē / payaṉpaṭutt-ātīrkaḷ`。
- 口語 `போகாதீங்க / பேசாதே` の社会的強さ。
- `-ஆதீர்கள்` を書記的すぎない主要表示として採用してよいか。

### `-லாம் / -லாமா`

- `நாம போகலாம்`：包含一人称による勧誘。
- `நாம போகலாமா?`：提案。
- `நான் உள்ளே போகலாமா?`：許可。
- `இங்கே உட்காரலாமா?`：許可の確認。

要確認：

- `-லாம்` の可能・推量までを第15課でどこまで触れるか。
- `நாம` がない `போகலாம்` の中立的な読み。
- `-லாமா` の許可・提案の場面差。

### はい・いいえ疑問

- `நீங்கள் வருகிறீர்களா?`：`varu-kiṟ-īrkaḷ-ā`
- `இந்த பஸ் சென்னைக்கு போகுமா?`：`pōk-um-ā`
- `இது உங்கள் டிக்கெட்டா?` のような名詞文疑問も可能。

要確認：

- `-ஆ` を独立した疑問接辞として示す分析。
- 名詞述語での表面融合 `டிக்கெட்டா`。
- 口語 `வரீங்களா / போகுமா` と完全形の距離。

### 疑問詞

- `யார்`：誰
- `என்ன`：何
- `எங்கே`：どこ
- `எப்போது`：いつ
- `ஏன்`：なぜ
- `எப்படி`：どうやって
- `எவ்வளவு`：どれくらい／いくら

疑問詞は知りたい位置に置き、英語型の語順反転は導入しない。

要確認：

- `ஏன் தாமதம்?` の訳を「なぜ遅い／遅れた」と幅を持たせること。
- `இங்கே எப்படி போகலாம்?` の自然さ。
- `எங்கே இறங்க வேண்டும்?` の旅行会話としての自然さ。

## 4. 第16課：名詞句

### 基本語順

教材上の入口は次の並びとする。

`指示・所有 ― 数・数量 ― 性質 ― 中心名詞 ― 複数 ― 格`

主要例：

- `அந்த இரண்டு புதிய தமிழ் படங்கள்`
- `என் புதிய புத்தகம்`
- `அந்த புதிய படத்தை`
- `இந்த இரண்டு டிக்கெட்டுகளை`

要確認：

- `அந்த இரண்டு புதிய தமிழ் படங்கள்` の自然な修飾順。
- `அந்த ஒரு புதிய படம்` が、語順展示用として不自然でないか。
- 所有者・指示語・数量を同時に置く場合の制約。

### 基本形容詞

- `நல்ல`：良い
- `பெரிய`：大きい
- `சின்ன`：小さい・口語
- `பழைய`：古い
- `புதிய`：新しい

形容詞は、中心名詞の性・数・格に一致して変化しないと説明する。

要確認：

- `சின்ன` を標準口語の中心語として置き、`சிறிய` を補足に回す方針。
- `நல்லவை` を「良いもの／作品」として第16課入口で使う難易度。

### `-ஆன` 派生形容表現

- `அழகு → அழகான`
- `நீளம் → நீளமான`
- `சூடு → சூடான`
- `மலிவு → மலிவான`

要確認：

- `aḻak-āṉa`、`nīḷam-āṉa`、`cūṭ-āṉa`、`maliv-āṉa` の形態境界。
- `சூடான தண்ணீர் / சூடான காபி` の温度ニュアンス。
- `மலிவான ஹோட்டல்` の自然さ。

### 数・数量・順序

- `ஒரு`：一つの／ある
- `இரண்டு`：二つの
- `சில`：いくつかの
- `நிறைய`：たくさんの
- `கொஞ்சம்`：少しの
- `முதல் / இரண்டாவது`：最初の／第二の

要確認：

- `இரண்டு டிக்கெட்டுகள் / இரண்டு புத்தகங்கள்` の複数語尾。
- 数詞後に単数形が選ばれる口語・語彙的条件。
- `நிறைய மக்கள்` と `சில நல்ல படங்கள்` の中立性。
- `முதல் நாள் / இரண்டாவது நாள்` の旅行語彙としての自然さ。

### 形容詞と副詞

- `வேகமான பஸ் / பஸ் வேகமாக போகிறது`
- `அழகான இடம் / இது அழகாக இருக்கிறது`

`-ஆன` は名詞修飾、`-ஆக` は動作・状態の様態という入口を示す。

要確認：

- `இது அழகாக இருக்கிறது` の自然な意味。
- `பஸ் வேகமாக போகிறது` の口語度。

### 比較・最上級的表現

- `இந்த வீடு அந்த வீட்டை விட பெரியது`
- `இந்த படம் அந்த படத்தை விட நீளமானது`
- `இந்த ஹோட்டல் அந்த ஹோட்டலை விட மலிவானது`
- `எல்லாவற்றிலும் இது மிகவும் மலிவானது`

比較の基準を対格形＋`விட` で示す。最上級は集合＋`மிகவும்` を中心にする。

要確認：

- 比較基準で対格 `-ஐ` が必要・自然な範囲。
- `விட` と口語 `விட`／`விடவும்` の差。
- `எல்லாவற்றிலும் இது மிகவும் மலிவானது` の自然さ。
- `இந்த மூன்றில் இது மிகவும் மலிவானது` の「最も」の解釈。

### 名詞省略と口語

- `பெரியது வேண்டும்`：大きいほうが欲しい。
- `சின்னது போதும்`：小さいので十分。
- `ரொம்ப நல்ல படம்`：とても良い映画。

要確認：

- 形容述語の名詞化 `பெரியது / சின்னது` を第16課で扱う範囲。
- `ரொம்ப` を中立的な映画口語として示す妥当性。

## 5. ローマ字と形態境界

主要例：

- `vā → vāruṅkaḷ`
- `pōk-ātē / pōk-ātīrkaḷ`
- `pōka-lām-ā`
- `varu-kiṟ-īrkaḷ-ā`
- `mukavari-ai-c colla muṭiyum-ā`
- `anta iraṇṭu putiya tamiḻ paṭaṅkaḷ`
- `anta putiya paṭatt-ai`
- `aḻak-āṉa iṭam`
- `vēkam-āka pō-kiṟ-atu`
- `anta hōṭṭal-ai viṭa malivāṉatu`

接合子音 `-c / -k / -p` は意味接辞として独立させず、表面形と構造表示の両立を確認する。

## 6. カタカナ上の注意

- `வாருங்கள் / பாருங்கள் / சொல்லுங்கள்` の `-ுங்கள்` を日本語で正確に再現できない。
- `போகாதே / போகாதீர்கள்` の長母音と歯茎音を近似に留める。
- `எவ்வளவு`、`அழகான`、`குறுகியது` は日本語話者が子音を脱落させやすい。
- `இரண்டு / இரண்டாவது` のそり舌音をカタカナで示しきれない。
- `விட` は日本語の「ヴィダ」と完全一致しない。

カタカナは音読開始用であり、正確な発音記号として評価しない。

## 9. 残る不確実性

第15課では、命令・丁寧依頼・勧誘の社会的距離と、口語縮約の地域差が最大の不確実性である。第16課では、数詞後の複数標識と、比較・最上級表現の自然な口語度が主要確認点となる。

現段階では次の二層方式を維持する。

- 主要例文：`வாருங்கள் / போகாதீர்கள் / போகலாமா / வருகிறீர்களா / அழகான / விட / மிகவும்`
- 文化カード：`வாங்க / பாருங்க / சொல்லுங்க / வரீங்களா / ரொம்ப`

第17・18課では、連体修飾と名詞化を扱う。名詞句の前に長い動詞句が置かれるため、第16課の「最後の名詞へまとめる」読み方をそのまま拡張する。
