# 言語検証メモ｜第1〜4課 増補版

画面へ表示しない内部確認用メモ。第5課以降の制作前に、母語話者またはタミル語学の専門家による確認を行う。

## 1. 採用するレジスター

- 中心は、チェンナイを含むタミル・ナードゥ州で広く理解される口語への入口。
- ただし動詞の主要例文では、TTSの安定性と形態素の透明性を優先し、`பார்க்கிறேன்`、`போகிறேன்`、`இருக்கிறேன்` など比較的保守的な綴りを残す。
- 映画会話で頻出する `பாக்கறேன் / பாக்குறேன்`、`போறேன்`、`இருக்கு` などの縮約形は、現段階では主要パラダイムへ混在させない。
- 代名詞では、口語性が高く教材価値の大きい `நாங்க`、`நாம`、`நீங்க`、`அவங்க` を明示的に口語形として採用する。
- 第3課では日常口語の `-ஓட -ōṭa` を中心形として採用する。
- 第4課では構造の透明な `-இல் -il`、`இடம் iṭam`、`இருந்து iruntu` を主要表示にし、`-ல`、`-கிட்ட` などは文化カード内の聞き取り補助に留める。

この方針は「完全な書記語」でも「特定地域のくだけた会話表記」でもない。教育用の二層レジスターであり、タミル文字は比較的保守的、文化カードと一部の語形は口語寄りとする。

## 2. 書記形・保守形との差

### 第1・2課

- `இதுக்கு it-ukku`：口語。書記形は `இதற்கு itaṟku`。
- `அதுக்கு at-ukku`：口語。書記形は `அதற்கு ataṟku`。
- `நாங்க nāṅka`：口語。書記形・保守形は `நாங்கள் nāṅkaḷ`。
- `நாம nāma`：口語。書記形・保守形は `நாம் nām`。
- `நீங்க nīṅka`：口語。書記形・保守形は `நீங்கள் nīṅkaḷ`。
- `அவங்க avaṅka`：口語の敬意単数／複数。書記形では文脈により `அவர்`、`அவர்கள்`。

### 第3課

- `என்னோட eṉṉōṭa`、`நண்பரோட naṇparōṭa`：口語の所有・関係／共同。
- 書記・改まった形では、所有に `-உடைய -uṭaiya`、共同に `-ஓடு -ōṭu`、`-உடன் -uṭaṉ` などが現れる。
- `-ஓட` を所有と共同の両方に使う説明は、標準的な口語の入口として採用。ただし地域・話者・文体による頻度差を確認する。
- `-ஆல் -āl` は構造の透明な形。実際の口語では末尾母音を伴うように聞こえる場合があるため、`-ஆல / -ஆலே` に近い音形の扱いを要確認。

### 第4課

- `வீட்டில் vīṭṭ-il`：主要表示。映画口語では `வீட்டுல` のような形が頻出する。
- `என்னிடம் eṉṉ-iṭam`：主要表示。口語では `என்கிட்ட` が頻出する。
- `வீட்டிலிருந்து vīṭṭ-il-iruntu`：主要表示。口語では `-லிருந்து`、`-லேர்ந்து` に近い縮約が聞かれる場合がある。
- 口語縮約は地域差と話速差が大きいため、現段階では音声再生対象の中心にしない。

字幕と台詞のレジスター差を文化説明に含めるが、「字幕は常に書記形」という意味ではない。作品・配信元・字幕方針による差がある。

## 3. ローマ字方式

- 長母音：`ā ī ū ē ō`
- そり舌音：`ṭ ṭṭ ṇ ḷ ḻ`
- 歯茎音等：`ṟ ṉ`
- 構造上の境界をハイフンで示す。
- ローマ字は純粋な音声転写ではなく、学習用の形態分析表示。
- 口語の表面音が構造形から大きく離れる場合だけ `surface` を併記する。

例：

- `vīṭu-kaḷ-il`：名詞―複数―場所格
- `kār-kaḷ-ai`：名詞―複数―対格
- `eṉṉ-ai`：一人称代名詞の対格用土台―対格
- `eṉ-akku`：一人称代名詞の斜格土台―与格
- `naṇpar-ōṭa`：名詞―口語の関係／共同標識
- `kai-y-āl`：名詞―接合子音―手段標識
- `vīṭṭ-il-iruntu`：斜格語幹―場所―起点
- `naṇpar-iṭam-iruntu`：人名詞―人の領域―起点
- `vīṭṭ-ukku-p pakkatt-il`：基準名詞―与格―接合子音―「近く」―場所

## 4. ハイフン分解の判断

### 斜格語幹

- `வீட்டில்`：`vīṭṭ-il`
  - 基本形 `vīṭu`
  - 格語尾前の接続形を `vīṭṭ-` として表示
- `படத்தில்`：`paṭatt-il`
  - 基本形 `paṭam`
  - 格語尾前の接続形を `paṭatt-` として表示
- `படத்தை`：`paṭatt-ai`

斜格語幹を独立して発音される語と誤解させない。説明では「語尾を受ける足場」と表現する。

### 表面形と構造形

- `புத்தகங்கள்`
  - 表面：`puttakaṅkaḷ`
  - 構造：`puttakam-kaḷ`
- `என்னோட`
  - 表面：`eṉṉōṭa`
  - 構造：`eṉṉ-ōṭa`
- `பஸ்ஸில்`
  - 表面：`passil`
  - 構造：`pas-il`

`surface` フィールドは、構造表示だけでは実際の形が見えにくい場合に限定する。

### 接合子音

`சென்னைக்குப் போகிறேன்`、`வீட்டுக்குப் பக்கத்தில்` などの `ப்` は、格語尾や語彙的接辞ではなく接合に関わる。第4課では完成形として表示するが、接合規則そのものは独立項目として未説明。

## 5. 第3課：所有・共同・手段

### `-ஓட -ōṭa`

- 所有・関係：`நண்பரோட போன்`「友達の電話」
- 共同：`நண்பரோட சேர்ந்து போகிறேன்`「友達と一緒に行く」
- 所有と共同の判定を、単独の訳語ではなく、後続する名詞・`சேர்ந்து`・動詞との関係で説明する。
- 共同例では曖昧さを避けるため `சேர்ந்து` を併用した。日常会話として過剰・不自然でないか要確認。

### `என் / உன்` と `என்னோட / உன்னோட`

- `என் வீடு`、`உன் போன்` を短い所有形として提示。
- `என்னோட வீடு`、`உன்னோட போன்` を口語で明示的な関係形として提示。
- 両者の語用差を「短い／明示的」だけで説明してよいか、焦点・対比・方言差を要確認。

### `-ஆல் -āl`

- `கையால் சாப்பிடுகிறேன்`：身体部位を手段として使う。
- `பேனாவால் எழுதுகிறேன்`：道具を手段として使う。
- 乗り物は `பஸ்ஸில் போகிறேன்` とし、日本語の「で」から `-āl` を選ばない対比を導入。
- 原因・受け身動作主への拡張は説明だけに留め、主要例文へ入れない。

## 6. 第4課：場所・移動

### 三方向

- 位置：`வீட்டில் இருக்கிறேன்`
- 到達点：`வீட்டுக்கு போகிறேன்`
- 起点：`வீட்டிலிருந்து வருகிறேன்`

`-il-iruntu` を「場所＋そこから」と分析する。共時的な学習分析として有効だが、歴史的語源説明として断定しない。

### `-il` と `iṭam`

- 物・場所：`வீட்டில்`、`கடையில்`
- 人のところ・手元：`நண்பரிடம்`、`என்னிடம்`
- `நண்பரிடம் ஒரு டிக்கெட் இருக்கிறது` を「友達は切符を持っている」と自然訳する。
- 有生複数、組織、抽象的領域などでの分布は単純な「物／人」二分を越えるため、第4課では代表的対照に限定する。

### 起点

- 場所：`கடையிலிருந்து`
- 人：`நண்பரிடமிருந்து`
- `நண்பரிடமிருந்து போன் வந்தது` が自然な保守形として適切か、映画口語との距離を確認する。

### 空間語

- `வீட்டுக்குள்`
- `வீட்டுக்கு வெளியே`
- `வீட்டுக்கு முன்னால்`
- `வீட்டுக்குப் பின்னால்`
- `வீட்டுக்குப் பக்கத்தில்`

第4課では完成表現として導入し、各語が要求する格・接合・語彙化の程度は第5課で整理する。

## 7. 対格・与格経験者・人称敬意（第1・2課から継続）

- 人・有生・特定された目的語では、対格 `-ai` の表示が重要。
- 不特定の無生物目的語では、特に口語で対格が現れない場合がある。
- `நான் படம் பார்க்கிறேன்` と `படத்தை நான் பார்க்கிறேன்` の差を単純な定・不定だけで断定しない。
- `எனக்கு காபி பிடிக்கும்`、`எனக்கு தண்ணீர் வேண்டும்` の説明で使う「届く先」は学習用メタファー。
- `அவன் / அவள்` は非敬意・親称として働き、`அவர் / அவங்க` は敬意を含む。
- 代名詞選択の含意は作品内の階層、地域、カースト、世代、人物造形により変わるため一律化しない。

## 8. カタカナで再現しにくい音

- `ழ ḻ`：日本語に直接対応する音がない。
- `ள ḷ`：そり舌の側面音。
- `ற ṟ`：位置と環境により実現が変わる。
- `ண ṇ`：そり舌鼻音。
- 母音間の `க ச ட த ப`：日本語カタカナでは有声・無声の中間的実現を安定して表せない。
- 語末の `ம் / ன் / ண்`：日本語の「ン」と同一ではない。
- `-ஓட`：長母音、そり舌音、語末母音を「オーダ／オード」のどちらへ寄せるか要統一。
- `இடம் iṭam` と口語 `கிட்ட kiṭṭa`：カタカナだけでは調音位置の差を示しにくい。

カタカナは音読開始用であり、正確な発音記号として評価しない。

## 9. 要ネイティブ確認

1. `போன் செய்கிறேன்` が教材用中立形として自然か。映画口語では `போன் பண்றேன்` が一般的な場面が多い可能性。
2. `அவர் / அவங்க` の敬意単数・複数の説明量。
3. `படம் பார்க்கிறேன் / படத்தை நான் பார்க்கிறேன்` の対比。
4. `-ஓட` を所有・共同の中心形として扱う範囲。
5. `நண்பரோட சேர்ந்து போகிறேன்` が中立口語として自然か。`சேர்ந்து` なしとの差。
6. `என் / என்னோட`、`உன் / உன்னோட` の語用差。
7. `கையால் / பேனாவால்` の主要表示と実際の口語音。
8. 「バスで」を `பஸ்ஸில்` とする例文・綴り・表面ローマ字。
9. `-il / iṭam` の基本対照を初学者向けに単純化しすぎていないか。
10. `வீட்டுல`、`என்கிட்ட`、起点形の口語縮約をどの課から主要音声へ入れるか。
11. `நண்பரிடமிருந்து போன் வந்தது`、`நண்பரிடம் போன் இருக்கிறது` の自然さ。
12. `வீட்டுக்கு முன்னால் / பின்னால் / பக்கத்தில்` の格選択と接合表記。
13. カタカナ表記、とくに `-ஓட`、`-ஆல்`、`இடம்`、`இலிருந்து`。
14. TTSが口語綴り `என்னோட`、`நண்பரோட` と保守形 `வீட்டிலிருந்து` を自然に読むか。
15. 保守的綴りと「標準口語」という教材ラベルの整合。

## 11. 残る不確実性

現状の最大の不確実性は、構造が透明な保守的綴りと、実際のチェンナイ口語音の距離である。第3・4課では次の二層方式を試している。

- 主要例文：形態構造を追える保守的綴り、または広く理解される口語形
- 文化カード：映画で遭遇しやすい縮約形と、基本形へ戻す聞き方

第5・6課を追加する前に、少なくとも `-ஓட`、`-ஆல்`、`-ல`、`-கிட்ட` の表示方針とTTS挙動を実機で確認する。
