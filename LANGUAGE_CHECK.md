# 言語検証メモ｜第1〜14課 増補版

画面へ表示しない内部確認用メモ。第15課以降の制作前に、母語話者またはタミル語学の専門家による確認を行う。

## 1. 教材レジスター

- 中心は、チェンナイを含むタミル・ナードゥ州で広く理解される口語への入口。
- 主要例文は形態構造とTamil TTSの安定性を優先し、比較的保守的な綴りを残す。
- 映画会話で頻出する `பாக்கறேன் / போறேன் / பாக்கல / பாத்துட்டேன் / வெச்சேன் / வேணும்` 等は文化カードへ分離する。
- 代名詞では `நாங்க / நாம / நீங்க / அவங்க`、所有では `-ஓட` を明示的な口語形として採用する。
- 第14課の主要活用表は `நீங்கள் / பார்க்கிறீர்கள்` 等の構造が見える形を使い、`நீங்க / பாக்கறீங்க` は聞き取り補足に置く。
- 主要表示と映画口語を無表示で混在させない。

## 2. 第1〜12課から継続する確認

### 名詞・格・後置詞

- `இதுக்கு / அதுக்கு` と書記形 `இதற்கு / அதற்கு`。
- `என் / என்னோட`、`உன் / உன்னோட` の語用差。
- `-ஓட` の所有・共同用法と `சேர்ந்து` の併用。
- `கையால் / பேனாவால்` と乗り物の `பஸ்ஸில்` の対照。
- `வீட்டில் / என்னிடம்` と口語 `வீட்டுல / என்கிட்ட` の距離。
- `படத்தைப் பற்றி / படம் பற்றி` の対格表示。
- `-க்காக / பதிலாக / தவிர` の格支配と接合子音。

### 名詞文・存在・経験者

- `இது என் வீடு` のゼロコピュラ説明。
- `இது என் வீடு இல்லை` と `வீடல்ல` 系の自然さ・分布。
- `என்னிடம் ... இருக்கிறது` と `எனக்கு ... இருக்கிறது` の所有表現上の重なり。
- `எனக்கு ... பிடிக்கும் / வேண்டும் / புரியும்` の与格経験者。

### 動詞・否定・非定形

- `பார்க்கிறேன் / பாக்கறேன் / பாக்குறேன்` の地域差。
- `பார்த்தேன்` を `pār-tt-ēṉ`、`போனேன்` を `pō-ṉ-ēṉ` と示す判断。
- `சாப்பிட்டேன்` を `cāppiṭṭ-ēṉ` とし、存在しない形態素を作らない方針。
- 単純現在による進行中・習慣の読み。
- `-வில்லை` の過去・未完了・現在不成立の読み分け。
- `மாட்டேன்` の未来否定・習慣・拒否の範囲。
- `என்னால் / எனக்கு ... முடியாது` の分布。
- `பார்→பார்க்க / வா→வர / செய்→செய்ய / வாங்கு→வாங்க` の不定詞表示。
- `வந்து / போய் / செய்து / பார்த்து / வாங்கி / சாப்பிட்டு` の副動詞表示。
- `-விட்டு / -கொண்டே / -ஆமல்` の意味範囲。
- `பார்த்துக்கொண்டிருக்கிறேன்` と単純現在の対比。
- `முடி / விடு / வை / பார்` の補助動詞用法。

## 3. 第13課：可能・希望・義務・意志

### 可能 `முடியும்`

- `என்னால் தமிழ் பேச முடியும்`：能力者を `-ஆல்` で示す。
- `இங்கே டிக்கெட் வாங்க முடியும்`：場所・条件としての可能。
- 否定は `முடியாது`。
- `மாட்டேன்` の意志的非実現、`கூடாது` の禁止と区別する。

要確認：

- 能力者として `என்னால்` を使う例の中立性。
- 口語で `எனக்கு ... முடியும்` が選ばれる範囲。
- `கார்டால் பணம் செலுத்த முடியும்` の語彙・自然さ。

### 希望 `ஆசை / வேண்டும் / விரும்பு`

- `எனக்கு சென்னைக்கு போக ஆசை இருக்கிறது`：願いを名詞として明示。
- `எனக்கு தண்ணீர் வேண்டும்`：物への欲求・必要。
- `போக வேண்டும்`：文脈により「行く必要がある」「行きたい」。
- `விரும்புகிறேன்` はより明示的・改まった希望として補足のみ。

要確認：

- `எனக்கு ... போக ஆசை இருக்கிறது` の標準口語としての自然さ。
- `நான் வீட்டுக்கு போக ஆசை இருக்கிறது` と `எனக்கு` の選択。
- `வேண்டும்` が希望として読まれる範囲と、義務として読まれる範囲。
- 口語 `வேணும் / போகணும்` の中立性と地域差。

### 必要・義務 `வேண்டும்`

- `நான் நாளை சீக்கிரம் வர வேண்டும்`：必要・義務。
- `பார்க்கவே வேண்டும்`：`-வே` による「どうしても／必ず」の強調。
- 日本語の「べき・ねばならない」と一対一に固定せず、状況・人間関係から強さを判断する。

要確認：

- `-வே வேண்டும்` の説明と例文の自然さ。
- 規則・個人的必要・助言の強度差をどこまで基礎編で示すか。

### 禁止 `கூடாது`

- `போன் பயன்படுத்தக் கூடாது`：規則・許可としての禁止。
- `இங்கே நிறுத்த முடியாது / இங்கே நிறுத்தக் கூடாது`：不可能と禁止の対照。
- 接合子音 `க்` を独立した意味接辞として扱わない。

要確認：

- `பயன்படுத்தக் கூடாது / பேசக் கூடாது / திறக்கக் கூடாது` の分かち書き。
- 日常口語での `கூடாது` と縮約形の距離。

### 意志・近接未来 `-ப் போகிறேன்`

- `டிக்கெட் வாங்கப் போகிறேன்`：移動目的または具体化した予定。
- `சொல்லப் போகிறேன்`：これから言う。
- `மழை பெய்யப் போகிறது`：目前の予測。
- 単純未来 `வாங்குவேன்` と、近接未来・計画の複合形を対比する。

要確認：

- 不定詞と `போகிறேன்` の間の `ப்` 接合。
- `மழை பெய்யப் போகிறது` の予測用法。
- 映画口語 `பாக்கப் போறேன் / சொல்லப் போறேன்` の自然さ。

## 4. 第14課：全人称

### 一・二人称

- 一人称単数：`-ēṉ`
- 一人称複数：`-ōm`
- 親しい二人称単数：`-āy`
- 丁寧単数・二人称複数：`-īrkaḷ`

主要例：

- `பார்க்கிறேன் / பார்த்தேன் / பார்ப்பேன்`
- `பார்க்கிறோம் / பார்த்தோம் / பார்ப்போம்`
- `பார்க்கிறாய் / பார்த்தாய் / பார்ப்பாய்`
- `பார்க்கிறீர்கள் / பார்த்தீர்கள் / பார்ப்பீர்கள்`

要確認：

- `நீங்கள்` を丁寧単数・複数の中心形として扱う説明。
- 口語 `நீங்க பார்க்கிறீங்க / பாக்கறீங்க` のローマ字表示。
- 親しい `நீ ... -āy` の語用説明が強すぎないか。

### 人間三人称

- 男性単数：`-āṉ`
- 女性単数：`-āḷ`
- 敬意単数：`-ār`
- 人間複数：`-ārkaḷ`

主要例：

- `அவன் பார்க்கிறான் / பார்த்தான் / பார்ப்பான்`
- `அவள் பார்க்கிறாள் / பார்த்தாள் / பார்ப்பாள்`
- `அவர் பார்க்கிறார் / பார்த்தார் / பார்ப்பார்`
- `அவர்கள் பார்க்கிறார்கள் / பார்த்தார்கள் / பார்ப்பார்கள்`

要確認：

- `அவர்` を性別非依存の敬意単数として説明する範囲。
- `அவர்கள்` の人間複数と、口語 `அவங்க` の敬意単数・複数両用。
- `-ārkaḷ` の形態分析とカタカナ。

### 非人間

- 単数現在：`வருகிறது varu-kiṟ-atu`
- 単数過去：`வந்தது vant-atu`
- 単数未来：`வரும் var-um`
- 複数現在：`வருகின்றன varu-kiṉṟ-aṉa`
- 複数過去：`வந்தன vant-aṉa`
- 複数未来：`வரும் var-um` を使用可能

非人間複数の現在で `-kiṉṟ-` が現れること、未来で単数・複数が同形になり得ることを、人間語尾表の単純な延長として扱わない。

要確認：

- `பஸ்கள் வருகின்றன / வந்தன / வரும்` の書記的自然さ。
- 標準口語では非人間複数一致がどこまで平準化されるか。
- `பஸ்கள் இங்கே நிற்கின்றன` の語形とTTS。
- `வருகிறது` のローマ字を `varu-kiṟ-atu` とする表示。

### 主語省略と聞き取り

- `பார்த்தாயா?`：二人称単数を末尾から復元。
- `பார்த்தார்`：敬意三人称。
- `வந்தது`：非人間単数。
- 口語では `பாக்கற / பாக்கறீங்க / பாக்கறான் / பாக்கறா / பாக்கறாங்க / வருது` を聞き取り補助にする。

要確認：

- `நீ பாக்கற` を標準的なチェンナイ口語の代表として置く妥当性。
- `அவ பாக்கறா` の代名詞縮約と女性語尾。
- `அவங்க பாக்கறாங்க` が敬意単数・人間複数の双方に使われる説明。
- 非人間複数口語を一つの固定表へしない方針。

## 5. ローマ字と形態境界

- 長母音：`ā ī ū ē ō`
- そり舌音：`ṭ ṭṭ ṇ ḷ ḻ`
- 歯茎音等：`ṟ ṉ`
- ハイフンは学習用形態分析であり、発音上の切れ目ではない。

主要例：

- `pēca muṭiyum`：話す接続形―可能
- `pōka ācai irukkiṟatu`：行く接続形―願い―存在
- `vara vēṇṭum`：来る接続形―必要
- `payaṉpaṭutta-k kūṭātu`：使う接続形―接合―禁止
- `vāṅka-p pō-kiṟ-ēṉ`：買う接続形―接合―行く現在―私
- `pār-kkiṟ-āy / pār-tt-āy / pār-pp-āy`
- `pār-kkiṟ-ār / pār-tt-ār / pār-pp-ār`
- `varu-kiṟ-atu / vant-atu / var-um`
- `varu-kiṉṟ-aṉa / vant-aṉa / var-um`

`-īrkaḷ / -ārkaḷ` を一つの人称終止語尾として示すか、敬意・複数要素へさらに分けるかは専門確認を行う。

## 6. カタカナ上の注意

- `வேண்டும் vēṇṭum` と口語 `வேணும் vēṇum / -ணும்` をカタカナで明確に区別しにくい。
- `கூடாது kūṭātu` の長母音・そり舌音を「クーダードゥ」で近似。
- `பார்க்கிறாய் / பார்க்கிறாள் / பார்க்கிறார்` は日本語では語末差が小さく聞こえる可能性がある。
- `அவன் / அவள் / அவர்` はカタカナでいずれも「アヴァル」に近づきやすく、Tamil文字とローマ字を優先する。
- `வருகின்றன` はカタカナが書記音へ寄り、実際の口語音と大きく離れる。

カタカナは音読開始用であり、正確な発音記号として評価しない。

## 9. 残る不確実性

第13課では `வேண்டும்` の希望・必要・義務の重なり、第14課では保守的な人称語尾表とチェンナイ口語の対応が最大の不確実性である。

現段階では次の二層方式を維持する。

- 主要例文：`முடியும் / ஆசை இருக்கிறது / வேண்டும் / கூடாது / -ப் போகிறேன்` と、完全な人称語尾を示す形
- 文化カード：`வேணும் / -ணும் / பாக்கறீங்க / பாக்கறாங்க / வருது` の映画口語

第15・16課では命令・疑問と名詞句を扱う。特に親しい命令・丁寧依頼・否定命令の社会的距離を、単なる活用表ではなく人物関係として示す必要がある。
