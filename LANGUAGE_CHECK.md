# 言語検証メモ｜第1〜8課 増補版

画面へ表示しない内部確認用メモ。第9課以降の制作前に、母語話者またはタミル語学の専門家による確認を行う。

## 1. 採用するレジスター

- 中心は、チェンナイを含むタミル・ナードゥ州で広く理解される口語への入口。
- 主要例文の動詞は、形態素の透明性とTamil TTSの安定性を優先し、`பார்க்கிறேன்`、`போகிறேன்`、`இருக்கிறேன்` など比較的保守的な綴りを残す。
- 映画会話で頻出する `பாக்கறேன் / பாக்குறேன்`、`போறேன்`、`இருக்கு / இருக்கேன்` などは、文化カード内の聞き取り補助に留める。
- 代名詞では、`நாங்க`、`நாம`、`நீங்க`、`அவங்க` を明示的な口語形として採用する。
- 第3課では口語の `-ஓட -ōṭa` を中心形として採用する。
- 第4課では `-இல் -il`、`இடம் iṭam`、`இருந்து iruntu` を主要表示にし、`-ல`、`-கிட்ட` などは補足に留める。
- 第5課では構造が透明な後置詞句を中心にし、`படம் பற்றி` のような短い口語形を対比例として示す。
- 第6課では名詞文・存在文・経験者文の構造を優先し、`இருக்கு`、`வேணும்` などは文化カード内に留める。
- 第7・8課では主要パラダイムを `-கிற்- / -க்கிற்-` 系の保守的綴りで統一し、口語縮約は別表示とする。

この方針は「完全な書記語」でも「特定地域のくだけた会話表記」でもない。主要表示は比較的保守的、代名詞・所有形・文化カードは口語寄りとする二層方式である。

## 2. 書記形・保守形との差

### 代名詞・格

- `இதுக்கு it-ukku`：口語。書記形は `இதற்கு itaṟku`。
- `அதுக்கு at-ukku`：口語。書記形は `அதற்கு ataṟku`。
- `நாங்க nāṅka`：口語。保守形は `நாங்கள் nāṅkaḷ`。
- `நாம nāma`：口語。保守形は `நாம் nām`。
- `நீங்க nīṅka`：口語。保守形は `நீங்கள் nīṅkaḷ`。
- `அவங்க avaṅka`：口語の敬意単数／複数。書記形では文脈により `அவர்`、`அவர்கள்`。
- `என்னோட eṉṉōṭa`、`நண்பரோட naṇparōṭa`：口語の所有・関係／共同。
- `வீட்டில் vīṭṭ-il` に対して口語 `வீட்டுல`、`என்னிடம் eṉṉ-iṭam` に対して口語 `என்கிட்ட` が頻出する。

### 後置詞・名詞文

- `படத்தைப் பற்றி paṭatt-ai-p paṟṟi`：構造が透明な完全形。会話では `படம் பற்றி paṭam paṟṟi` も頻出する。
- `நண்பருக்காக naṇpar-ukkāka` は学習上 `naṇpar-ukku + āka` と分析する。
- `இருக்கிறது irukkiṟatu`：主要表示。口語では `இருக்கு irukku` が頻出する。
- `வேண்டும் vēṇṭum`：主要表示。口語では `வேணும் vēṇum` に近く聞こえる。
- `இது என் வீடு இல்லை` を口語的な名詞文否定として採用したが、`இது என் வீடல்ல` 等との分布を要確認。

### 動詞

- `பார்க்கிறேன் pārkkiṟēṉ`：主要表示。口語では `பாக்கறேன் / பாக்குறேன்` が聞かれる。
- `போகிறேன் pōkiṟēṉ`：主要表示。口語では `போறேன் pōṟēṉ` に近い形が聞かれる。
- `பார்க்கிறோம் pārkkiṟōm`：主要表示。口語では `பாக்கறோம் / பாக்குறோம்` が聞かれる。
- 過去・未来は現在形より構造差が小さい場合もあるが、母音・子音の実現は口語で変化する。

字幕と台詞のレジスター差を文化説明に含めるが、「字幕は常に書記形」という意味ではない。作品・配信元・字幕方針による差がある。

## 3. ローマ字方式

- 長母音：`ā ī ū ē ō`
- そり舌音：`ṭ ṭṭ ṇ ḷ ḻ`
- 歯茎音等：`ṟ ṉ`
- 構造上の境界をハイフンで示す。
- ローマ字は純粋な音声転写ではなく、学習用の形態分析表示。
- 口語の表面音が構造形から大きく離れる場合だけ `surface` または文化補足を用いる。

例：

- `vīṭu-kaḷ-il`：名詞―複数―場所格
- `eṉṉ-ai`：代名詞土台―対格
- `naṇpar-ōṭa`：名詞―口語の関係／共同標識
- `vīṭṭ-il-iruntu`：斜格語幹―場所―起点
- `paṭatt-ai-p paṟṟi`：斜格語幹―対格―接合子音―「について」
- `pār-kkiṟ-ēṉ`：動詞の核―現在形を作る部分―一人称単数
- `pār-tt-ēṉ`：動詞の核―過去形を作る部分―一人称単数
- `pār-pp-ēṉ`：動詞の核―未来形を作る部分―一人称単数
- `pō-ṉ-ōm`：動詞の核―この動詞の過去形―一人称複数
- `cāppiṭṭ-ēṉ`：完成した過去幹―一人称単数

## 4. ハイフン分解の判断

### 名詞・接合子音

- `வீட்டில்`：`vīṭṭ-il`、基本形 `vīṭu`。
- `படத்தில்`：`paṭatt-il`、基本形 `paṭam`。
- `படத்தைப் பற்றி` の `ப்`、`அவரைத் தவிர` の `த்`、`வீட்டுக்குப் பக்கத்தில்` の `ப்` は意味を持つ格語尾として数えない。
- `புத்தகங்கள்` は表面 `puttakaṅkaḷ`、構造 `puttakam-kaḷ`。

### 動詞

- `பார்க்கிறேன்`：`pār-kkiṟ-ēṉ`。
- `பார்த்தேன்`：`pār-tt-ēṉ`。
- `பார்ப்பேன்`：`pār-pp-ēṉ`。
- `படிக்கிறேன்`：`paṭi-kkiṟ-ēṉ`。
- `படித்தேன்`：`paṭi-tt-ēṉ`。
- `படிப்பேன்`：`paṭi-pp-ēṉ`。
- `செய்தேன்`：`cey-t-ēṉ`。表面の `ceytēṉ` と形態境界を区別する。
- `போனேன்`：`pō-ṉ-ēṉ` と分析する。過去標識を独立した万能接辞として一般化しない。
- `சாப்பிட்டேன்`：`cāppiṭṭ-ēṉ` とし、存在しない `cāppiṭu-ṭṭ-` を作らない。
- `சாப்பிடுவேன்`：`cāppiṭu-v-ēṉ`。

現在標識は動詞クラスや接続により `-kiṟ- / -kkiṟ-` と見える。教材では完成形に合わせて表示し、全動詞へ同一の文字列を機械的に貼らない。

## 5. 第3〜6課の継続確認

- `-ஓட` を所有・共同の中心形として扱う範囲。
- `என் / என்னோட`、`உன் / உன்னோட` の語用差。
- `கையால் / பேனாவால்` と乗り物の `பஸ்ஸில்` の対照。
- `-il / iṭam` と口語 `-ல / -கிட்ட` の距離。
- `படத்தைப் பற்றி / படம் பற்றி` の対比。
- `-க்காக / பதிலாக / தவிர` の格支配と接合表記。
- `இது என் வீடு இல்லை` と `வீடல்ல` 系の分布。
- `என்னிடம்` と `எனக்கு` の所有表現上の重なり。

## 6. 第7課：動詞の設計図

### 動詞テンプレート

教材上の基本テンプレートは次のとおり。

`動詞の核―時制を作る部分―人称・数・性の末尾`

これは学習用の一般化であり、全動詞が完全に同じ三接辞へ機械的に分かれるという断定ではない。動詞クラス、音韻交替、融合した過去幹を考慮する。

### 動詞クラス

- `செய்`：`செய்கிறேன் / செய்தேன் / செய்வேன்`
- `பார்`：`பார்க்கிறேன் / பார்த்தேன் / பார்ப்பேன்`
- `படி`：`படிக்கிறேன் / படித்தேன் / படிப்பேன்`
- `சாப்பிடு`：`சாப்பிடுகிறேன் / சாப்பிட்டேன் / சாப்பிடுவேன்`

クラス番号一覧は導入せず、頻出動詞の三時制セットからパターンを見せる。`சாப்பிட்டேன்` では過去幹を一まとまりとして扱う。

### 自動詞・他動詞

- `போகிறேன்`：直接目的語を要求しない代表例。
- `படம் பார்க்கிறேன்`：見る動作が対象へ向かう代表例。

意味分類だけでなく、Tamil文内で対格目的語を取るかを観察する。自他動詞対・使役は後続課へ送る。

### 命令形・核

- `வா / போ / பார் / படி / சாப்பிடு / கேள்` を親しい単数への命令として提示。
- 「動詞の核に近い形」と説明し、日本語の辞書形・Tamil不定詞・丁寧依頼と同一視しない。
- `கேள்` の現在・過去・未来活用は複雑なため、第7課では命令形だけに留める。

## 7. 第8課：一人称の三時制

### 人称末尾

- 一人称単数：`-ēṉ`
- 一人称複数：`-ōm`

両末尾は現在・過去・未来で共通する。`நாங்க / நாம` の包含差は代名詞が担い、動詞末尾はいずれも基本的に `-ōm`。

### 現在

- `இப்போது சாப்பிடுகிறேன்`：進行中。
- `தினமும் காபி குடிக்கிறேன்`：習慣。

単純現在を進行専用と説明しない。明示的進行構文は第12課へ送る。

### 過去

- `பார்த்தேன்`、`போனேன்`、`சாப்பிட்டேன்`、`படித்தேன்`、`செய்தேன்`。
- 動詞ごとに過去幹の作り方が違うため、一つの過去接辞だけで説明しない。

### 未来

- `பார்ப்பேன்`、`போவேன்`、`சாப்பிடுவேன்`、`படிப்பேன்`、`செய்வேன்`。
- 一人称未来は予測だけでなく、意志・予定・約束を自然に表す。
- `நாம நாளை படம் பார்ப்போம்` は包含一人称＋未来形が勧誘として読まれる例。中立的な未来と勧誘の境界は第15課で再説明する。

### 主語省略

- `இன்று வீட்டில் இருக்கிறேன்`、`நாளை வருவேன்` のように、主語代名詞を置かなくても末尾から一人称単数を復元できる。
- 口語縮約でも `-ēṉ / -ōm` が聞き取りの手掛かりとしてどの程度安定するか、実音声で確認する。

## 8. カタカナで再現しにくい音

- `ழ ḻ`、`ள ḷ`、`ற ṟ`、`ண ṇ` は日本語に直接対応しない。
- 母音間の `க ச ட த ப` は有声・無声の単純二分で表せない。
- `பார்க்கிறேன்` の `ற`、`பார்த்தேன்` の重子音、`பார்ப்பேன்` の `ப்ப` をカタカナで区別しきれない。
- `போனேன்` の鼻音と長母音、`செய்தேன்` の連続子音を日本語音へ寄せすぎる可能性がある。
- `இன்று iṉṟu` を「インル」としたが、実際の口語音との差を要確認。

カタカナは音読開始用であり、正確な発音記号として評価しない。

## 9. 要ネイティブ確認

1. `பார்க்கிறேன் / பாக்கறேன் / பாக்குறேன்` の中立性・地域差。
2. `போகிறேன் / போறேன்`、`போகிறோம் / போறோம்` の対応。
3. `pār-kkiṟ-ēṉ`、`paṭi-kkiṟ-ēṉ` のハイフン判断。
4. `cey-t-ēṉ`、`pō-ṉ-ēṉ`、`cāppiṭṭ-ēṉ` の分析表示。
5. `பார் / படி / சாப்பிடு` を動詞の核・親しい命令形として導入する説明。
6. `கேள்` のカタカナと、命令形単独での自然さ。
7. `பஸ் இங்கே நிற்கிறது` の語形・ローマ字 `niṟ-kiṟ-atu`。
8. 単純現在による進行中・習慣の例文が中立的か。
9. 一人称未来を意志・予定として説明する範囲。
10. `நாம நாளை படம் பார்ப்போம்` の勧誘解釈と口語自然性。
11. `இன்று வீட்டில் இருக்கிறேன்`、`நாளை வருவேன்` の主語省略例。
12. `இன்று`、`நேற்று`、`நாளை` のカタカナ近似。
13. Tamil TTSが三時制の子音対立と `-ēṉ / -ōm` を聞き分けられる品質で読むか。
14. 8課分の連続再生で、短い命令形と三語連続TTSが不自然にならないか。
15. 保守的綴りと「標準口語」という教材ラベルの整合。

## 11. 残る不確実性

第7・8課で最大の問題は、形態構造が透明な保守的現在形と、実際のチェンナイ口語現在形の距離である。現段階では次の二層方式を採用する。

- 主要例文：`பார்க்கிறேன் / போகிறேன்` 型で形態構造を示す
- 文化カード：`பாக்கறேன் / பாக்குறேன் / போறேன்` 型を聞き取り補助として示す

第9・10課では否定形と不定詞を扱う。肯定現在形の単純な反転では作れないため、口語否定・書記否定・時制解釈を一度に混ぜない設計が必要である。
