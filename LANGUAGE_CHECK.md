# 言語検証メモ｜全20課 試作完成版

画面へ表示しない内部確認用メモ。全20課の横断修正前に、母語話者またはタミル語学の専門家による確認を行う。

## 1. 教材レジスター

- 中心は、チェンナイを含むタミル・ナードゥ州で広く理解される口語への入口。
- 主要例文は形態構造とTamil TTSの安定性を優先し、比較的保守的な綴りを残す。
- 映画口語は文化カードへ分離し、主要パラダイムと無表示で混在させない。
- 代名詞では `நாங்க / நாம / நீங்க / அவங்க`、所有では `-ஓட` を明示的な口語形として採用する。
- 主要表示は構造が見える形、文化欄は会話音へ戻す形、という二層方式を維持する。
- 全20課完成後も「承認済み」ではない。第1・2課を見本として実機・母語話者確認し、その修正を全課へ反映する。

## 2. 第1〜18課から継続する主要確認

### 名詞・格・後置詞

- `இதுக்கு / அதுக்கு` と保守的な `இதற்கு / அதற்கு`。
- `என் / என்னோட`、`உன் / உன்னோட` の語用差。
- `-ஓட` の所有・共同用法と `சேர்ந்து` の併用。
- `கையால் / பேனாவால்` と乗り物の `பஸ்ஸில்`。
- `வீட்டில் / என்னிடம்` と口語 `வீட்டுல / என்கிட்ட`。
- `படத்தைப் பற்றி / படம் பற்றி` の格表示。
- `-க்காக / பதிலாக / தவிர` の格支配と接合子音。

### 名詞文・存在・経験者

- `இது என் வீடு` のゼロコピュラ。
- `இது என் வீடு இல்லை` と `வீடல்ல` 系の距離。
- `என்னிடம் ... இருக்கிறது` と `எனக்கு ... இருக்கிறது` の所有表現上の重なり。
- `எனக்கு ... பிடிக்கும் / வேண்டும் / புரியும்` の与格経験者。

### 動詞・否定・非定形

- `பார்க்கிறேன் / பாக்கறேன் / பாக்குறேன்` の地域差。
- `பார்த்தேன்` を `pār-tt-ēṉ`、`போனேன்` を `pō-ṉ-ēṉ` と示す判断。
- `சாப்பிட்டேன்` を `cāppiṭṭ-ēṉ` とし、存在しない形態素を作らない方針。
- 単純現在による進行中・習慣の読み。
- `-வில்லை` の過去・未完了・現在不成立。
- `மாட்டேன்` の未来否定・習慣・拒否。
- `முடியாது` の能力・状況としての不可能。
- `பார்→பார்க்க / வா→வர / செய்→செய்ய / வாங்கு→வாங்க` の不定詞表示。
- `வந்து / போய் / செய்து / பார்த்து / வாங்கி / சாப்பிட்டு` の副動詞表示。
- `-விட்டு / -கொண்டே / -ஆமல்` の意味範囲。
- `பார்த்துக்கொண்டிருக்கிறேன்` と単純現在の対比。
- `முடி / விடு / வை / பார்` の補助動詞用法。

### モダリティ・全人称

- `முடியும் / முடியாது` と能力者 `என்னால் / எனக்கு` の分布。
- `ஆசை இருக்கிறது / வேண்டும் / விரும்புகிறேன்` の希望表現の距離。
- `வேண்டும் / -வே வேண்டும் / கூடாது` の強さ。
- `-ப் போகிறேன்` の移動目的・近接未来・意志・予測。
- `-āy / -īrkaḷ / -āṉ / -āḷ / -ār / -ārkaḷ` の三時制対応。
- `அவர் / அவர்கள்` と口語 `அவங்க` の敬意・複数。
- 非人間単数 `வருகிறது / வந்தது / வரும்`。
- 非人間複数 `வருகின்றன / வந்தன / வரும்` と口語での一致平準化。

### 命令・疑問・名詞句

- `வா / வாருங்கள் / வாங்க` の距離・縮約・多義性。
- `கொஞ்சம் ... முடியுமா?` の依頼としての丁寧さ。
- `-ஆதே / -ஆதீர்கள்` の否定命令。
- `-லாம் / -லாமா` の勧誘・許可・可能性。
- はい・いいえ疑問の `-ஆ` と名詞述語での融合。
- `அந்த இரண்டு புதிய தமிழ் படங்கள்` の修飾語順。
- 数詞後の複数語尾。
- `-ஆன / -ஆக` の形態分析。
- 対格形＋`விட`、集合＋`மிகவும்` の比較表現。

### 連体修飾・名詞化

- `பார்க்கிற / பார்த்த / பார்க்கும் / பார்க்காத` の主要表示。
- `-உம்` 連体形の未来・習慣・一般性。
- 中心名詞が連体節内で主語・目的語・場所・相手となる例。
- 長い連体名詞句への対格・与格・処格。
- `பார்ப்பது / பார்த்தது / பார்க்காதது` の名詞化表示。
- 「もの」と「出来事・事実」の読み分け。
- `சொன்னதை / வந்ததை / பார்ப்பதற்கு` の格形。
- `என்பது` の導入量と書記的 `-தல்` の位置づけ。
- 口語 `பாக்கற படம் / வர்ற ஆள் / பாக்கறது / பாத்தது` の地域差。

## 3. 第19課：引用と発言

### `என்று` の基本機能

主要例：

- `அவர் நாளை வருவேன் என்று சொன்னார்`
- `இந்த படம் நல்லது என்று நினைக்கிறேன்`
- `அவர் வந்தார் என்று கேள்விப்பட்டேன்`
- `நீ எங்கே போகிறாய் என்று அவர் கேட்டார்`

`என்று` の左側を引用内容としてまとめ、右側の発言・思考・質問動詞へ渡すと説明する。

要確認：

- `என்று` を引用標識・補文標識のどこまでの用語で説明するか。
- `அவர் நாளை வருவேன் என்று சொன்னார்` で、引用符なしに一人称を保持する自然さ。
- `நாளை / இங்கே / நான்` 等の直示表現が元の発話時点を保つ説明。
- 直接引用と内容報告を基礎編でどこまで厳密に区別するか。

### 発言・思考・伝聞

- `சொன்னேன் / சொன்னார்`：言った
- `நினைக்கிறேன் / நினைத்தேன்`：思う／思った
- `கேள்விப்பட்டேன்`：聞き知った
- `கேட்டேன் / கேட்டார்`：聞いた／尋ねた

要確認：

- `கேட்டேன்` が「尋ねた」と「聞いた」の双方を表す範囲。
- 伝聞に `கேள்விப்பட்டேன்` を中心形として選ぶ自然さと語彙難度。
- `நண்பர் படம் நல்லது என்று சொன்னார்` の主語・発言内容の自然さ。
- `அந்த படம் நீளமானது என்று நினைத்தேன்` の形容述語。

### 質問の引用

- `எங்கே போகிறாய் என்று கேட்டார்`
- `எப்போது வருவாய் என்று கேட்டார்`
- `வருவாயா என்று கேட்டார்`
- `வாங்கினாயா என்று கேட்டார்`

疑問詞または `-ஆ` を引用内容内に残す。`என்று` 自体を疑問標識とは説明しない。

要確認：

- 埋込み疑問で疑問語尾・イントネーションをどこまで保つか。
- `டிக்கெட் இருக்கிறதா என்று அவர் கேட்டார்` の自然さ。
- 疑問詞質問に主語を明示・省略する差。

### 命令と指示

- `சீக்கிரம் வா என்று சொன்னார்`：命令文を直接引用
- `சீக்கிரம் வரச் சொன்னார்`：来るように指示
- `போகாதே என்று சொன்னார்`：否定命令を直接引用
- `போக வேண்டாம் என்று சொன்னார்`：禁止・不要の内容

要確認：

- `வரச் சொன்னார் / கதவை மூடச் சொன்னார்` の接合子音・目的語関係。
- `வேண்டாம்` の禁止・不要・拒否の範囲。
- 直接引用に引用符を画面上で使わない方針。

### `என்பது` との対比

- `இந்த படம் நல்லது என்பது என் கருத்து`
- `இந்த படம் நல்லது என்று நினைக்கிறேன்`

前者は内容を名詞として扱い、後者は内容を思考動詞へ渡すという入口にする。

要確認：

- `என்பது` と `என்று` の統語的関係を単純化しすぎていないか。
- 名詞化・引用・補文の用語を初学者向けにどこまで使うか。

### 映画口語

- `என்று → -ன்னு / -னு`
- `வருவேன் என்று சொன்னான் → வருவேன்னு சொன்னான்`
- `நினைத்தேன் → நினைச்சேன்`
- `சொன்னார்கள் → சொன்னாங்க`

要確認：

- `-ன்னு / -னு` の地域差・音韻条件。
- 引用内容末尾と `-ன்னு` の融合をローマ字でどう示すか。
- `நினைச்சேன்` を映画口語の代表として置く妥当性。

## 4. 第20課：条件・理由・目的・時間

### 条件 `-ஆல்`

主要例：

- `நேரம் இருந்தால்`
- `நண்பர் வந்தால்`
- `பார்த்தால்`
- `டிக்கெட் கிடைத்தால்`

過去報告ではなく、未実現条件・一般条件にも使うと説明する。

要確認：

- `வரு → வந்தால் / பார் → பார்த்தால் / கிடை → கிடைத்தால்` の完成形表示。
- 条件形を過去語幹＋`-ஆல்` と分析する説明の程度。
- `நண்பர் வந்தால் போகலாம்` の自然さ。

### 否定条件

- `டிக்கெட் இல்லையென்றால்`
- `நீ வரவில்லை என்றால்`
- `நீ வராவிட்டால்`

透明な否定文＋`என்றால்` から、まとまった否定条件へ進む。

要確認：

- `இல்லையென்றால்` の分かち書き・ローマ字 `illai-eṉṟāl`。
- `வரவில்லை என்றால்` と `வராவிட்டால்` の語用差。
- 口語 `வரலனா / இல்லனா / இல்லேனா` の中立性・地域差。

### 譲歩 `-ஆலும்`

- `மழை பெய்தாலும் போவேன்`
- `படம் நீளமாக இருந்தாலும் பார்க்கலாம்`

単純条件 `-ஆல்` と、条件を認めても結果が変わらない `-ஆலும்` を対比する。

要確認：

- `பெய்தாலும்` を未来的な「降っても」に使う説明。
- `நல்லதாக இருந்தால் / நீளமாக இருந்தாலும்` の形容状態。

### 理由 `-தால்`

- `வேலை இருப்பதால்`
- `மழை பெய்ததால்`
- `அவர் வராததால்`
- `அதனால் / ஏனென்றால்`

要確認：

- `இருப்பது＋ஆல் → இருப்பதால்`、`பெய்தது＋ஆல் → பெய்ததால்` の教育的分析。
- ローマ字 `iruppat-āl / peytat-āl / varātat-āl`。
- `மழை பெய்ததால் ரயில் தாமதமாக வந்தது` の自然さ。
- `ஏனென்றால்` と口語 `ஏன்னா` の範囲。

### 目的

- `படம் பார்க்க தியேட்டருக்கு போகிறோம்`
- `படம் பார்ப்பதற்காக தியேட்டருக்கு போகிறோம்`
- `நண்பரை பார்க்க சென்னைக்கு வந்தேன்`
- `நண்பரை பார்ப்பதற்காக சென்னைக்கு வந்தேன்`

要確認：

- 移動目的の短い不定詞が標準口語として自然か。
- `-தற்காக` の明示性・書記性。
- `பார்ப்பதற்காக` の形態表示 `pārppataṟkāka`。

### 時間節

- `நண்பர் வரும்போது`
- `படம் பார்க்கும்போது`
- `வருவதற்கு முன்`
- `வந்த பிறகு`
- `நண்பர் வரும் வரை`

要確認：

- `வரும்போது` の「来るとき／来たら」の範囲。
- `பார்க்கும்போது` の同時進行。
- `வருவதற்கு முன்` と `வரும் முன்` の距離。
- `வந்த பிறகு` と口語 `வந்தப்புறம்`。
- `வரும் வரை` の継続終点。
- `படம் தொடங்கிய பிறகு` の過去連体形。

### 長文解体

代表文：

`மழை பெய்தாலும், டிக்கெட் இருந்தால், நண்பர் வரும்போது நாம தியேட்டருக்கு போகலாம்.`

分析順：

1. 最後の有限述語 `போகலாம்`
2. 時間節 `நண்பர் வரும்போது`
3. 条件節 `டிக்கெட் இருந்தால்`
4. 譲歩節 `மழை பெய்தாலும்`

要確認：

- 三つの従属節を重ねた例文の自然さ・情報密度。
- コンマがTTSへ与える影響。
- 学習用の節順が実際の会話で自然か。

### 映画口語

- `வந்தால் → வந்தா`
- `இருந்தால் → இருந்தா`
- `இல்லையென்றால் → இல்லனா / இல்லேனா`
- `ஏனென்றால் → ஏன்னா`
- `அதனால் → அதனால`
- `வரும்போது → வர்றப்போ`
- `வந்த பிறகு → வந்தப்புறம்`

要確認：

- 条件 `-ஆ` と疑問 `-ஆ` の聞き分け。
- `-ன்னா` が条件・理由・引用由来で現れる場合の説明。
- `-ப்போ / -ப்புறம்` の地域差。

## 5. ローマ字と形態境界

第19・20課の主要例：

- `varu-v-ēṉ eṉṟu coṉṉ-ār`
- `nallatu eṉṟu niṉaikkiṟ-ēṉ`
- `varuvāy-ā eṉṟu kēṭṭ-ār`
- `vara-c coṉṉ-ār`
- `irunt-āl / peyt-ālum`
- `illai-eṉṟāl / varāviṭṭāl`
- `iruppat-āl / peytat-āl`
- `pārppataṟkāka`
- `var-um-pōtu / varuvataṟku muṉ / vanta piṟaku / var-um varai`

接合子音・融合形は意味接辞として捏造せず、表面形と学習用境界を併記する。

## 6. カタカナ上の注意

- `என்று / என்றால் / ஏனென்றால்` の `ṉṟ` をカタカナで再現できない。
- `சொன்னார் / கேட்டார் / கேள்விப்பட்டேன்` の重子音・そり舌音を近似に留める。
- `இருந்தால் / பெய்தாலும் / இருப்பதால்` の語末差がカタカナでは弱くなる。
- `வருவதற்கு / பார்ப்பதற்காக` の長い形を日本語音節へ過剰に分割しない。
- 口語 `-ன்னு / -ன்னா / -ப்போ / -ப்புறம்` はTamil文字と実音声を優先する。

カタカナは音読開始用であり、正確な発音記号として評価しない。
