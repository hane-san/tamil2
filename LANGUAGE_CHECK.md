# 言語検証メモ｜第1〜18課 増補版

画面へ表示しない内部確認用メモ。第19・20課の制作前に、母語話者またはタミル語学の専門家による確認を行う。

## 1. 教材レジスター

- 中心は、チェンナイを含むタミル・ナードゥ州で広く理解される口語への入口。
- 主要例文は形態構造とTamil TTSの安定性を優先し、比較的保守的な綴りを残す。
- 映画会話で頻出する `பாக்கறேன் / போறேன் / பாக்கல / பாத்துட்டேன் / வெச்சேன் / வேணும் / பாருங்க / பாக்கறது` 等は文化カードへ分離する。
- 代名詞では `நாங்க / நாம / நீங்க / அவங்க`、所有では `-ஓட` を明示的な口語形として採用する。
- 主要表示は構造が見える形、文化欄は会話音へ戻す形、という二層方式を維持する。

## 2. 第1〜16課から継続する確認

### 名詞・格・後置詞

- `இதுக்கு / அதுக்கு` と書記形 `இதற்கு / அதற்கு`。
- `என் / என்னோட`、`உன் / உன்னோட` の語用差。
- `-ஓட` の所有・共同用法と `சேர்ந்து` の併用。
- `வீட்டில் / என்னிடம்` と口語 `வீட்டுல / என்கிட்ட` の距離。
- `படத்தைப் பற்றி / படம் பற்றி` の対格表示。
- `-க்காக / பதிலாக / தவிர` の格支配と接合子音。

### 名詞文・存在・経験者

- `இது என் வீடு` のゼロコピュラ説明。
- `இது என் வீடு இல்லை` と `வீடல்ல` 系の自然さ・分布。
- `என்னிடம் ... இருக்கிறது` と `எனக்கு ... இருக்கிறது` の所有表現上の重なり。
- `எனக்கு ... பிடிக்கும் / வேண்டும் / புரியும்` の与格経験者。

### 動詞・否定・非定形

- `பார்க்கிறேன் / பாக்கறேன் / பாக்குறேன்` の地域差。
- `பார்த்தேன்` を `pār-tt-ēṉ`、`போனேன்` を `pō-ṉ-ēṉ` と示す判断。
- `சாப்பிட்டேன்` を `cāppiṭṭ-ēṉ` とし、存在しない形態素を作らない方針。
- 単純現在による進行中・習慣の読み。
- `-வில்லை` の過去・未完了・現在不成立の読み分け。
- `மாட்டேன்` の未来否定・習慣・拒否の範囲。
- `என்னால் / எனக்கு ... முடியாது` の分布。
- `பார்→பார்க்க / வா→வர / செய்→செய்ய / வாங்கு→வாங்க` の不定詞表示。
- `வந்து / போய் / செய்து / பார்த்து / வாங்கி / சாப்பிட்டு` の副動詞表示。
- `-விட்டு / -கொண்டே / -ஆமல்` の意味範囲。
- `பார்த்துக்கொண்டிருக்கிறேன்` と単純現在の対比。
- `முடி / விடு / வை / பார்` の補助動詞用法。

### モダリティ・全人称

- `முடியும் / முடியாது` と能力者 `என்னால்` の分布。
- `ஆசை இருக்கிறது / வேண்டும் / விரும்புகிறேன்` の希望表現の距離。
- `வேண்டும் / -வே வேண்டும் / கூடாது` の強さ。
- `-ப் போகிறேன்` の移動目的・近接未来・意志・予測。
- `-āy / -īrkaḷ / -āṉ / -āḷ / -ār / -ārkaḷ` の三時制対応。
- `அவர் / அவர்கள்` と口語 `அவங்க` の敬意・複数。
- 非人間単数 `வருகிறது / வந்தது / வரும்`。
- 非人間複数 `வருகின்றன / வந்தன / வரும்` と口語での一致平準化。

### 命令・疑問・名詞句

- `வா / வாருங்கள் / வாங்க` の距離・縮約・多義性。
- `கொஞ்சம் ... முடியுமா?` の依頼としての丁寧さ。
- `-ஆதே / -ஆதீர்கள்` の否定命令。
- `-லாம் / -லாமா` の勧誘・許可・可能性。
- はい・いいえ疑問の `-ஆ` と名詞述語での融合。
- `அந்த இரண்டு புதிய தமிழ் படங்கள்` の修飾語順。
- 数詞後の複数語尾。
- `-ஆன / -ஆக` の形態分析。
- 対格形＋`விட`、集合＋`மிகவும்` の比較表現。

## 3. 第17課：連体修飾

### 基本四形

教材では `பார்` を中心に次を対比する。

- `பார்க்கிற படம்`：`pār-kkiṟa paṭam`――現在・進行に関わる映画
- `பார்த்த படம்`：`pār-tta paṭam`――見た映画
- `பார்க்கும் படம்`：`pārkk-um paṭam`――これから見る／普段見る映画
- `பார்க்காத படம்`：`pārkk-āta paṭam`――見ない／見ていない映画

連体形は人称語尾を持たず、節内の主語または文脈から行為者を判断する。

要確認：

- `பார்க்கிற` を、標準口語への入口として主要表示に置く妥当性。
- より書記的な `பார்க்கின்ற` を補足しない方針。
- `பார்த்த` の形態境界を `pār-tta` とする表示。
- `பார்க்காத` が現在・過去・習慣の否定へ広がる説明。

### 主語と中心名詞

主要例：

- `நான் நேற்று பார்த்த படம்`
- `அவர் சொன்ன கதை`
- `நேற்று வந்த நண்பர்`
- `சென்னைக்கு போகும் பஸ்`
- `நான் இன்னும் படிக்காத புத்தகம்`

中心名詞は連体節内で主語・目的語・到達点・相手など異なる役割を持ち得る。独立した関係代名詞は置かず、動詞と文脈から関係を補う。

要確認：

- `நான் பேசும் நண்பர்` を「私が話す相手」とする自然さ。
- `வருகிற அவர்` より `வருகிற நண்பர் / வருகிறவர்` が自然か。
- `நான் போகும் ஹோட்டல்` と `நாம் தங்கும் ஹோட்டல்` の語用差。
- 連体節内で到達点・相手の格が表面に出ない例を基礎編でどこまで扱うか。

### `-உம்` の範囲

- `நாளை பார்க்கும் படம்`：未来の予定
- `அடிக்கடி பார்க்கும் படம்`：習慣
- `சென்னைக்கு போகும் பஸ்`：予定された行き先／一般的性質

`-um` を未来専用とせず、非過去・習慣・一般的性質を名詞へつなぐ形として説明する。

### 長い名詞句と格

- `நான் பார்த்த படத்தை`
- `நேற்று வந்த நண்பருக்கு`
- `நான் போகும் ஹோட்டலில்`
- `நான் பார்க்காத படத்துக்கு டிக்கெட்`

格語尾は連体形ではなく、最後の中心名詞へ付く。

要確認：

- `paṭam → paṭatt-ai / paṭatt-ukku` の形態表示。
- `நான் பார்க்காத படத்துக்கு டிக்கெட் வாங்குகிறேன்` の自然さ。
- 長い名詞句で接合音をどこまでローマ字へ示すか。

### 映画口語

- `பார்க்கிற படம் → பாக்கற படம்`
- `வருகிற ஆள் → வர்ற ஆள்`
- `போகும் பஸ் → போற பஸ்`
- `பார்த்த படம் → பாத்த படம்`

要確認：

- `-ற` を現在連体形の一律な口語標識として扱いすぎていないか。
- `போற பஸ்` が予定・習慣を表す中立的なチェンナイ口語か。
- `சொன்ன விஷயம் / சொன்னது` の教材上の対比。

## 4. 第18課：名詞化

### 基本三形

- `செய்வது / செய்தது / செய்யாதது`
- `பார்ப்பது / பார்த்தது / பார்க்காதது`
- `வருவது / வந்தது / வராதது`

教材では、非過去・一般、過去・完了、否定の三組として示す。単一の接辞付加規則に還元せず、頻出動詞の完成形で覚えさせる。

要確認：

- `பார்ப்பது` と連体形 `பார்க்கும்` の形態関係を、初学者へどこまで説明するか。
- `வருவது` のローマ字を `varuvatu` と一続きにする判断。
- `செய்யாதது` を `ceyy-ātatu` と分ける判断。
- 否定名詞化が現在・過去の両方へ広がる説明。

### 連体修飾との対比

- `நான் பார்த்த படம்`：私が見た映画
- `நான் பார்த்தது`：私が見たもの／見たこと
- `நான் படம் பார்த்தது`：私が映画を見たという出来事
- `நான் பார்க்காத படம்`：私が見ていない映画
- `நான் பார்க்காதது`：私が見ていないもの／見なかったこと

要確認：

- `பார்த்தது` を「もの」と「事実」の双方へ訳す説明。
- 節内に目的語がある場合、出来事読みが強まるという説明の妥当性。
- headless relative と clausal nominalization を基礎編で同一形として扱う範囲。

### 主語・目的語としての名詞化

- `தமிழ் படிப்பது சுவாரஸ்யம்`
- `எனக்கு தமிழ் படம் பார்ப்பது பிடிக்கும்`
- `அவர் நேற்று வந்தது எனக்குத் தெரியும்`
- `அவர் வராதது எனக்கு கவலை`
- `அவர் சொன்னதை புரிந்துகொண்டேன்`

要確認：

- `தமிழ் படிப்பது சுவாரஸ்யம்` の自然さと語彙難度。
- `அவர் வராதது எனக்கு கவலையாக இருக்கிறது / எனக்கு கவலை` の自然さ。
- `தெரியும்` の内容に対格が表面化しない説明。
- `புரிந்துகொண்டேன்` と `புரிந்தது` の語用差。

### 名詞化＋格

- `சொன்னது → சொன்னதை`
- `வந்தது → வந்ததை`
- `பார்ப்பது → பார்ப்பதற்கு`
- `சொன்னதைப் பற்றி`

要確認：

- `து＋ஐ → தை` の表面変化を初学者へどう示すか。
- `பார்ப்பதற்கு` を `pārppataṟku` とするローマ字表示。
- `அவர் வந்ததை பார்த்தேன்` が「来るのを見た／来たことを確認した」のどこまでを含むか。
- `படம் பார்ப்பதற்கு நேரம் இல்லை` が不定詞目的表現とどう重なるか。

### `என்பது`

- `அவர் நாளை வருவார் என்பது உறுதி`
- `இந்த படம் நல்லது என்பது என் கருத்து`
- `தமிழ் கடினம் என்பது உண்மை அல்ல`

完成した有限文・名詞文を「〜ということ・事実」として包む。第19課の `என்று`・引用動詞へ接続する。

要確認：

- `என்பது` を第18課で独立した名詞化標識として導入する順序。
- `அவர் வருவது உறுதி` と `அவர் வருவார் என்பது உறுதி` の意味差。
- `தமிழ் கடினம் என்பது உண்மை அல்ல` の自然さと断定の強さ。

### 書記形 `-தல்`

- `படித்தல் / செல்லுதல்` 等は、書面・見出し・規則文で増える。
- 口語中心の主要表示へ混ぜず、補足に留める。

要確認：

- `-தல்` を行政・法律文に限りすぎない説明。
- `படிப்பது` と `படித்தல்` のレジスター差。

### 映画口語

- `பார்ப்பது → பாக்கறது`
- `வருவது → வர்றது`
- `செய்வது → செய்றது`
- `பார்த்தது → பாத்தது`
- `பார்க்காதது → பாக்காதது`

要確認：

- `பாக்கறது` を主要形 `பார்ப்பது` へ直接対応させる説明。
- 口語では連体形＋中性代名詞として再分析できる範囲。
- `பாத்தது` の「見たもの／見た事実」の文脈依存。

## 5. ローマ字と形態境界

主要例：

- `pār-kkiṟa paṭam`
- `pār-tta paṭam`
- `pārkk-um paṭam`
- `pārkk-āta paṭam`
- `nāṉ pārtta paṭatt-ai`
- `ceyvatu / ceytatu / ceyy-ātatu`
- `pārppatu / pārttatu / pārkkātatu`
- `avar coṉṉat-ai`
- `paṭam pārppataṟku`
- `avar nāḷai varu-v-ār eṉpatu`

連体形・名詞化形の境界は、歴史的分析より学習者が現在の形を再利用できることを優先する。存在しない語幹・接辞は作らない。

## 6. カタカナ上の注意

- `பார்க்கிற / பார்க்கும் / பார்க்காத` の子音連続をカタカナで区別しきれない。
- `வாங்கிய / சொன்ன / வந்த` の連体形は実際には後続名詞と密接に連続する。
- `பார்ப்பது / பார்ப்பதற்கு` の重子音とそり舌音を正確に表せない。
- `என்பது` の鼻音・有声性を「エンバドゥ」で近似する。
- 口語 `பாக்கறது / வர்றது` はカタカナへ固定せず、Tamil文字と音声を優先する。

カタカナは音読開始用であり、正確な発音記号として評価しない。

## 9. 残る不確実性

第17課では、保守的な連体形と映画口語 `-ற` の対応、中心名詞が節内で果たす役割の説明が主要確認点である。第18課では、同じ `-து` 系が「もの」と「出来事・事実」を表す範囲、格付き名詞化、`என்பது` の導入量が最大の不確実性である。

現段階では次の二層方式を維持する。

- 主要例文：`பார்க்கிற படம் / பார்த்த படம் / பார்க்கும் படம் / பார்க்காத படம் / பார்ப்பது / பார்த்தது / என்பது`
- 文化カード：`பாக்கற படம் / வர்ற ஆள் / பாக்கறது / பாத்தது / பாக்காதது`

第19・20課では、引用と条件・理由・時間節を扱う。第18課の `என்பது` から `என்று` へつなぎ、最後は複数の節を「最後の有限動詞」から逆算する読み方へ統合する。
