# 言語検証メモ｜第1〜12課 増補版

画面へ表示しない内部確認用メモ。第13課以降の制作前に、母語話者またはタミル語学の専門家による確認を行う。

## 1. 教材レジスター

- 中心は、チェンナイを含むタミル・ナードゥ州で広く理解される口語への入口。
- 主要例文は形態構造とTamil TTSの安定性を優先し、比較的保守的な綴りを残す。
- 映画会話で頻出する `பாக்கறேன் / போறேன் / பாக்கல / பாத்துட்டேன் / வெச்சேன்` 等は文化カードへ分離する。
- 代名詞では `நாங்க / நாம / நீங்க / அவங்க`、所有では `-ஓட` を明示的な口語形として採用する。
- 主要表示と映画口語を無表示で混在させない。

## 2. 第1〜10課から継続する確認

### 名詞・格・後置詞

- `இதுக்கு / அதுக்கு` と書記形 `இதற்கு / அதற்கு`。
- `என் / என்னோட`、`உன் / உன்னோட` の語用差。
- `-ஓட` の所有・共同用法と `சேர்ந்து` の併用。
- `கையால் / பேனாவால்` と乗り物の `பஸ்ஸில்` の対照。
- `வீட்டில் / என்னிடம்` と口語 `வீட்டுல / என்கிட்ட` の距離。
- `படத்தைப் பற்றி / படம் பற்றி` の対格表示。
- `-க்காக / பதிலாக / தவிர` の格支配と接合子音。

### 名詞文・存在・経験者

- `இது என் வீடு` のゼロコピュラ説明。
- `இது என் வீடு இல்லை` と `வீடல்ல` 系の自然さ・分布。
- `என்னிடம் ... இருக்கிறது` と `எனக்கு ... இருக்கிறது` の所有表現上の重なり。
- `எனக்கு ... பிடிக்கும் / வேண்டும் / புரியும்` の与格経験者。

### 肯定・否定・不定詞

- `பார்க்கிறேன் / பாக்கறேன் / பாக்குறேன்` の地域差。
- `பார்த்தேன்` を `pār-tt-ēṉ`、`போனேன்` を `pō-ṉ-ēṉ` と示す判断。
- `சாப்பிட்டேன்` を `cāppiṭṭ-ēṉ` とし、存在しない形態素を作らない方針。
- 単純現在による進行中・習慣の読み。
- `-வில்லை` の過去・未完了・現在不成立の読み分け。
- `மாட்டேன்` の未来否定・習慣・拒否の範囲。
- `என்னால் / எனக்கு ... முடியாது` の分布。
- `பார்→பார்க்க / வா→வர / செய்→செய்ய / வாங்கு→வாங்க` の不定詞表示。
- `வரச் சொல்லுங்கள்` の接合表記と丁寧さ。

## 3. 第11課：副動詞

### 基本形

- `வா → வந்து`：`vā → vantu`
- `போ → போய்`：`pō → pōy`
- `செய் → செய்து`：`cey → ceytu`
- `பார் → பார்த்து`：`pār → pārttu`
- `குடி → குடித்து`：`kuṭi → kuṭittu`
- `வாங்கு → வாங்கி`：`vāṅku → vāṅki`
- `சாப்பிடு → சாப்பிட்டு`：`cāppiṭu → cāppiṭṭu`

副動詞形成を単一接辞の機械的追加として教えず、頻出動詞の完成ペアで示す。

### 有限動詞との関係

- `போன் செய்து கேட்கிறேன்`：最後の `கேட்கிறேன்` が一人称現在を担う。
- `சாப்பிட்டு வீட்டுக்கு போனேன்`：食べたのも行ったのも通常同じ人物として読む。
- 基礎的な副動詞列では主語共有を説明するが、絶対的な統語制約として断定しない。

要確認：

- `கேட்கிறேன்` のローマ字を `kēṭ-kiṟ-ēṉ` と示す判断。
- `சிரித்து பேசுகிறார்` が「笑って／笑いながら話す」の代表例として自然か。
- `கடைக்கு போய் தண்ணீர் வாங்கி வருகிறேன்` の中立性。

### `-விட்டு`

- `பார்த்துவிட்டு போனேன்`：前の動作を完了・一区切りしてから次へ進む。
- 本動詞 `விடு` の「放す・置いていく」と、補助動詞的な完了・離脱の関係を説明する。
- 第12課の完了補助動詞へ接続する。

要確認：

- `பார்த்து-விட்டு` と `பார்த்துவிட்டு` のローマ字・形態境界。
- `சாப்பிட்டுவிட்டு வருகிறேன்` の自然な意味範囲。
- 映画口語 `பாத்துட்டு / போயிட்டு` の地域差。

### `-கொண்டே`

- `பேசிக்கொண்டே நடக்கிறேன்`：話す動作を続けたまま歩く。
- `நடந்து கொண்டே பேசுகிறோம்` の分かち書きと `நடந்துகொண்டே` の一語表記の選択を要確認。
- 同時進行と継続強調の境界を単純化しすぎない。

### `-ஆமல்`

- `சாப்பிடாமல் வந்தேன்`：食べずに来た。
- `பார்க்காமல் போனேன்`：見ずに行った。
- `பார்க்க போகவில்லை`：見に行かなかった。

否定が前の動作へ掛かるか、移動全体へ掛かるかを比較する。

要確認：

- `cāppiṭ-āmal / pārkk-āmal` の形態分析表示。
- `பார்க்காமல் வீட்டுக்கு போனேன்` の教材例としての自然さ。

## 4. 第12課：補助動詞

### 進行 `கொண்டு இரு`

- `பார்த்துக்கொண்டிருக்கிறேன்`：`pārttu-kkoṇṭu-irukkiṟ-ēṉ`
- `படித்துக்கொண்டிருக்கிறேன்`：`paṭittu-kkoṇṭu-irukkiṟ-ēṉ`
- `சாப்பிட்டுக்கொண்டிருக்கிறேன்`：`cāppiṭṭu-kkoṇṭu-irukkiṟ-ēṉ`

単純現在 `பார்க்கிறேன்` と、進行を明示する複合形を対比する。両者の意味領域は重なるため、単純現在を非進行専用とは説明しない。

要確認：

- 主要表示の分かち書き・綴り `பார்த்துக்கொண்டிருக்கிறேன்`。
- 口語 `பாத்துக்கிட்டு இருக்கேன்` の中立性・地域差。
- 長い形をTTSが自然に読むか。

### 継続 `-கொண்டே இரு`

- `பேசிக்கொண்டே இருக்கிறார்`：話し続けている。
- `பார்த்துக்கொண்டே இருக்கிறேன்`：見続けている。
- 進行形との意味差を「途中／途切れず続ける」という目安で示すが、文脈上の重なりを残す。

### 完了 `முடி`

- `படித்து முடித்தேன்`：読み終えた。
- `வேலை செய்து முடித்தேன்`：仕事を終えた。
- `முடி` が終点への到達を明示する代表例として自然か確認する。

### 完了・既済・態度 `விடு`

- `பார்த்துவிட்டேன்`：もう見た／見てしまった。
- `மறந்துவிட்டேன்`：忘れてしまった。
- `போனை வீட்டில் விட்டுவிட்டேன்`：電話を家へ置いてきてしまった。

`விடு` 自体を「後悔」の語と説明せず、完了・既済から文脈的に安堵・意外・残念などが生じるとする。

要確認：

- `போனை வீட்டில் விட்டுவிட்டேன்` の自然さと二つの `விடு` の分析。
- `படம் பார்த்து முடித்துவிட்டோம்` の補助動詞列が教材例として過剰でないか。
- 口語 `பாத்துட்டேன் / மறந்துட்டேன் / சொல்லிட்டேன்` の縮約分析。

### 結果保持・準備 `வை`

- `டிக்கெட் வாங்கி வைத்தேன்`：切符を買っておいた。
- `தண்ணீர் வாங்கி வைத்தோம்`：水を買っておいた。
- 口語 `வெச்சேன்` と保守形 `வைத்தேன்` の距離を確認する。

### 試行 `பார்`

- `கேட்டுப் பார்த்தேன்`：試しに尋ねてみた。
- `சாப்பிட்டுப் பார்த்தேன்`：試しに食べてみた。
- 本動詞 `பார்`「見る」と補助動詞「試す」を、前の副動詞の有無から区別する。

要確認：

- `கேட்டுப் பார்த்தேன்` の接合子音 `ப்`。
- `இந்த உணவை சாப்பிட்டுப் பார்த்தேன்` の自然さ。
- 試行用法が書記的すぎず、標準口語への入口として適切か。

## 5. ローマ字と形態境界

- 長母音：`ā ī ū ē ō`
- そり舌音：`ṭ ṭṭ ṇ ḷ ḻ`
- 歯茎音等：`ṟ ṉ`
- ハイフンは学習用形態分析であり、発音上の切れ目ではない。

主要例：

- `ceytu kēṭ-kiṟ-ēṉ`：して―尋ねる―現在―私
- `pārttu-viṭṭu pō-ṉ-ēṉ`：見て―完了・離脱―行く過去―私
- `pēci-kkoṇṭē naṭa-kkiṟ-ēṉ`：話し―継続―歩く―現在―私
- `cāppiṭ-āmal vant-ēṉ`：食べずに―来た―私
- `pārttu-kkoṇṭu-irukkiṟ-ēṉ`：見て―保ち―いる現在―私
- `paṭittu muṭi-tt-ēṉ`：読んで―終える過去―私
- `pārttu-viṭṭ-ēṉ`：見て―完了・既済―私
- `vāṅki vai-tt-ēṉ`：買って―保持する過去―私
- `kēṭṭu-p pār-tt-ēṉ`：尋ねて―接合―試す過去―私

`கொண்டு / விட்டு / வைத்து` を独立語・補助動詞・融合した複合形のどこで分けるかは、学習上の透明性と実際の綴りの両方を確認する。

## 6. カタカナ上の注意

- `செய்து ceytu` を「セイドゥ」としたが、母音間子音と連続音を完全には表せない。
- `போய் pōy` の二重母音を「ポーイ」で近似する。
- `பார்த்துக்கொண்டிருக்கிறேன்` はカタカナが長くなり、音節境界を誤解させる可能性がある。
- `முடித்தேன் muṭittēṉ`、`வைத்தேன் vaittēṉ` の重子音と二重母音を日本語で再現しきれない。
- 口語 `பாத்துட்டேன் / வெச்சேன்` はカタカナへ固定せず、Tamil文字と音声を優先する。

## 9. 残る不確実性

第11・12課では、副動詞・補助動詞の境界と口語での融合が最大の不確実性である。現段階では次の二層方式を維持する。

- 主要例文：`செய்து / பார்த்துவிட்டு / பார்க்காமல் / பார்த்துக்கொண்டிருக்கிறேன் / வாங்கி வைத்தேன்` の構造が見える形
- 文化カード：`செஞ்சு / பாத்துட்டு / பாக்காம / பாத்துக்கிட்டு இருக்கேன் / வெச்சேன்` の映画口語

第13・14課では可能・希望・義務と全人称を扱う。特に口語の人称語尾と、保守的な主要表示の対応を表形式で過度に詰め込まない設計が必要である。
