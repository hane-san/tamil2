# 言語検証メモ｜第1〜10課 増補版

画面へ表示しない内部確認用メモ。第11課以降の制作前に、母語話者またはタミル語学の専門家による確認を行う。

## 1. 教材レジスター

- 中心は、チェンナイを含むタミル・ナードゥ州で広く理解される口語への入口。
- 主要例文は形態構造とTamil TTSの安定性を優先し、`பார்க்கிறேன்`、`போகிறேன்`、`இருக்கிறது` など比較的保守的な綴りを残す。
- 映画会話で頻出する `பாக்கறேன் / பாக்குறேன்`、`போறேன்`、`இருக்கு`、`பாக்கல` などは文化カード内に分離する。
- 代名詞では `நாங்க / நாம / நீங்க / அவங்க`、所有では `-ஓட` を明示的な口語形として採用する。
- 主要表示と映画口語を無表示で混在させない。

## 2. 継続して確認する語形

### 名詞・格・後置詞

- `இதுக்கு / அதுக்கு` と書記形 `இதற்கு / அதற்கு`。
- `என்னோட / உன்னோட` と短い所有形 `என் / உன்` の語用差。
- `-ஓட` の所有・共同用法と `சேர்ந்து` の併用。
- `கையால் / பேனாவால்` と、乗り物の `பஸ்ஸில்` の対照。
- `வீட்டில் / என்னிடம்` と口語 `வீட்டுல / என்கிட்ட` の距離。
- `படத்தைப் பற்றி / படம் பற்றி` の対格表示。
- `நண்பருக்காக / காபிக்குப் பதிலாக / அவரைத் தவிர` の格支配と接合子音。

### 名詞文・存在・経験者

- `இது என் வீடு` のゼロコピュラ説明。
- `இது என் வீடு இல்லை` と `இது என் வீடல்ல` 系の自然さ・分布。
- `என்னிடம் ... இருக்கிறது` と `எனக்கு ... இருக்கிறது` の所有表現上の重なり。
- `எனக்கு ... பிடிக்கும் / வேண்டும் / புரியும்` の与格経験者。

### 肯定動詞

- `பார்க்கிறேன் / பாக்கறேன் / பாக்குறேன்` の地域差・中立性。
- `போகிறேன் / போறேன்`、`போகிறோம் / போறோம்` の対応。
- `பார்த்தேன்` を `pār-tt-ēṉ`、`போனேன்` を `pō-ṉ-ēṉ` と示す判断。
- `சாப்பிட்டேன்` を `cāppiṭṭ-ēṉ` とし、存在しない形態素を作らない方針。
- 単純現在による進行中・習慣の読み。
- 一人称未来による意志・予定・約束と、`-ōm` の勧誘解釈。

## 3. 第9課：否定

### `-வில்லை`

教材では次を中心にする。

- `நேற்று படம் பார்க்கவில்லை`：過去の不成立「昨日見なかった」
- `இன்னும் படம் பார்க்கவில்லை`：未完了「まだ見ていない」
- `இப்போது படம் பார்க்கவில்லை`：現在の不成立「今は見ていない」

`-வில்லை` 型は人称語尾を持たないものとして説明する。`நான் பார்க்கவில்லை / அவர் பார்க்கவில்லை` で、否定述語自体は同じである。

要確認：

- `இப்போது சாப்பிடவில்லை` が中立的に「今は食べていない」を表す範囲。
- 完了・過去・現在不成立の読みを、時間語だけで説明して単純化しすぎていないか。
- 口語 `பாக்கல / போகல / சாப்பிடல` の地域差・時制解釈。

### `மாட்டேன் / மாட்டோம்`

- `பார்க்க மாட்டேன்`：見ない、見るつもりがない、拒否する。
- `பொதுவாக காபி குடிக்க மாட்டேன்`：習慣的に飲まない。
- `நாளை போக மாட்டேன்`：未来の予定・意志として行かない。
- `மாட்டேன்` には `-ēṉ`、`மாட்டோம்` には `-ōm` がある。

要確認：

- 習慣否定としての `பொதுவாக ... மாட்டேன்` の自然さ。
- 文脈なしで `மாட்டேன்` が持つ拒否・意志の強さ。
- 映画字幕で「絶対しない」「嫌だ」等に訳される場合の説明量。

### `முடியாது`

- `என்னால் தமிழ் பேச முடியாது`：能力者を `-ஆல்` で示す。
- `இங்கே கார் நிறுத்த முடியாது`：一般的・状況的な不可能。
- `மாட்டேன்` と `முடியாது` を、意志的非実現と能力・状況的不可能として対比する。

要確認：

- 能力者を `என்னால்` で示す例の中立性。
- 日常口語での `எனக்கு ... முடியாது` との分布。
- `புரிந்துகொள்ள முடியாது` の語形・カタカナ・難易度。

### その他の否定

- `தெரியாது` を頻出完成形として示す。
- `teriy-ātu` の分解を初学者向けにどこまで見せるか要確認。
- `வேண்டாம்`、否定命令、`-ஆமல்` は後続課へ送る。

## 4. 第10課：不定詞

### 基本ペア

- `பார் → பார்க்க`：`pār → pārkka`
- `போ → போக`：`pō → pōka`
- `வா → வர`：`vā → vara`
- `படி → படிக்க`：`paṭi → paṭikka`
- `சாப்பிடு → சாப்பிட`：`cāppiṭu → cāppiṭa`
- `செய் → செய்ய`：`cey → ceyya`
- `பேசு → பேச`：`pēcu → pēca`
- `வாங்கு → வாங்க`：`vāṅku → vāṅka`

不定詞形成を単一接辞の機械的追加として教えず、頻出動詞の完成ペアで示す。

要確認：

- `சாப்பிட` のローマ字を `cāppiṭa` とする学習分析。
- `வாங்க` の基底を `vāṅku`、不定詞を `vāṅka` と示す判断。
- `சொல்ல`、`உட்கார`、`நிறுத்த` などの不定詞表示。

### 目的・助動詞への接続

- `படம் பார்க்க போகிறேன்`：目的＋移動。
- `பேச முடியும்`：可能。
- `வர வேண்டும்`：必要・義務。
- `போக மாட்டேன்`：非実現・拒否。
- `வரச் சொல்லுங்கள்`：内容・指示。

不定詞自体は時制・人称を完成させず、最後の有限動詞・述語が文を完成させると説明する。

要確認：

- `படம் பார்க்க தியேட்டருக்கு போகிறேன்` と `படம் பார்க்கப் போகிறேன்` の接合・語用差。
- `நண்பரை பார்க்க போகிறேன்` を「友達に会いに行く」と訳す自然さ。
- `வரச் சொல்லுங்கள்` の `ச்`、丁寧形 `சொல்லுங்கள்`、教材導入時期。
- `பாக்கப் போறேன்` を映画口語の代表として出す中立性。

### 文化補足

- 南インドで `ஹோட்டல்` が食堂・レストランを指す用法は実在するが、地域・店の種類・英語運用による差がある。
- `சாப்பிட ஹோட்டலுக்கு போகிறார்` が自然な教材例かネイティブ確認する。

## 5. ローマ字とハイフン

- 長母音：`ā ī ū ē ō`
- そり舌音：`ṭ ṭṭ ṇ ḷ ḻ`
- 歯茎音等：`ṟ ṉ`
- ハイフンは学習用形態分析であり、発音上の切れ目ではない。

主要例：

- `pār-kkiṟ-ēṉ`：見る―現在―私
- `pār-tt-ēṉ`：見る―過去―私
- `pār-pp-ēṉ`：見る―未来―私
- `pārkka māṭṭ-ēṉ`：見る不定詞―非実現―私
- `pārkka-villai`：見る不定詞―不成立
- `pārkka muṭiyātu`：見る不定詞―不可能
- `eṉṉ-āl pēca muṭiyātu`：私―手段格―話す不定詞―不可能
- `vara-c colluṅkaḷ`：来る不定詞―接合子音―言ってください

`-வில்லை` を `-villai` と一続きに表示するか、`-v-illai` とさらに分けるかは要専門確認。現段階では初学者向けに `pārkka-villai` とする。

## 6. カタカナ上の注意

- `மாட்டேன் māṭṭēṉ`：長母音・そり舌重子音を「マーッテーン」で近似。
- `முடியாது muṭiyātu`：母音間子音を「ムディヤードゥ」で近似。
- `பாக்கல`：口語音をカタカナだけで固定しない。
- `வரச் சொல்லுங்கள்`：`ச் சொ` の連結音を「チ　ソ」ではなく「チソ／ッチョ」に近く感じる話者もあり得る。
- `ஹோட்டல் hōṭṭal`：英語 hotel 由来だが、Tamil内の音価を英語発音と同一視しない。

カタカナは音読開始用であり、正確な発音記号として評価しない。

## 9. 残る不確実性

第9・10課では、否定の時間解釈と口語縮約が最大の不確実性である。現段階では次の二層方式を維持する。

- 主要例文：`பார்க்கவில்லை / பார்க்க மாட்டேன் / பார்க்க முடியாது / பார்க்க போகிறேன்` の構造が見える形
- 文化カード：`பாக்கல / போகல / பாக்கப் போறேன்` の映画口語

第11・12課では副動詞と補助動詞を扱うため、`-ஆமல்`、`-க்கொண்டு`、`-விட்டு` 等をどこまで一つの構文・補助動詞として分解するかを改めて検討する。
