# 口語タミル語の教科書

チェンナイ口語タミル語を、一人称単数から対話、第三者、長文化、旅行10場面、文法補完へ組み上げる静的教材アプリです。第1〜38章、502例文、167問の診断問題、旅行語彙262項目、文法87型、活用表、端末音声、A5印刷表示を収録しています。v2.8では、既存38章の前に独立した PART 0 を加え、名詞格の早見、対格 `-ai / -e / Ø` の判断、12母音×18子音の音声文字表を収録しました。例文カードは丸い再生ボタンだけでなく、カード内のどこを押しても再生できます。

## すぐ使う

1. `index.html` をブラウザで開く。
2. 最初に PART 0 で名詞格と文字音を確認し、各章では画面下の「読む → 例文 → 形／語彙 → 練習」の順に進む。
3. 例文カード内のどこか、または丸い再生ボタンを押して個別再生。「全例文を聞く」で章を連続再生。
4. 右上の `Aa` からカナ・直訳・語の切れ目を表示／非表示にする。

左上の章一覧では、PART 0 へいつでも戻れます。章名・文法・タミル語・日本語語彙による検索、既存7部の絞り込み、未完了だけの表示、次の未完了章への再開、章ごとの5ステップ進捗も利用できます。

ファイルを直接開いても本文・練習・音声読み上げは使えます。ホーム画面へのインストールと確実なオフラインキャッシュには、HTTPSまたはローカルHTTPサーバーで開いてください。

ローカルサーバー例：

```bash
python3 -m http.server 8000
```

その後、ブラウザで `http://localhost:8000/` を開きます。

## 音声

- 音声にはブラウザの `speechSynthesis` と端末内のタミル語音声（`ta-IN`）を使います。
- TTSへ送る文字列はタミル文字だけです。書記形と口語音がずれる項目では、表示用 `ta` ではなく音声用 `ttsText` を使います。日本語、ローマ字、カナ、文法記号は読み上げません。
- PART 0 の文字表では、母音見出し、子音見出し、12×18の各交点を一文字ずつ読み上げます。
- 通常／ゆっくり、1回／2回、前／次、停止、連続再生があります。
- 音質とオフライン可否は端末のタミル語音声に依存します。旅行前に機内モードで試してください。

## 印刷

右上 `Aa` から「この章を印刷」または「PART 0＋全38章を印刷」を選びます。印刷CSSはA5縦、例文・語彙・文法カードを高密度で配置し、操作部品を隠します。PDF保存にも使えます。

## ファイル

- `index.html` — 画面の骨格
- `styles.css` — モバイル・A5印刷レイアウト
- `curriculum.js` — 教材の正本データ
- `travel-curriculum.js` — 第19〜28章の旅行科データと語彙台帳
- `grammar-curriculum.js` — 第29〜38章の文法補完科データと文法型台帳
- `reference-curriculum.js` — PART 0 の名詞格早見、目的語標示、タミル文字表
- `transcription-revision.js` — 書記形から独立した口語ローマ字・カナ・TTS読みと本文内注記
- `app.js` — ナビ、進捗、練習、音声、印刷
- `manifest.webmanifest` / `sw.js` — インストール・オフラインキャッシュ
- `tamil_verb_engine_v2_standalone.html` — 1ファイルで開ける配布版
- `PRODUCTION_GUIDE_V2.md` — 今後の全章を同じ思想で増築する制作規格
- `LANGUAGE_CHECK.md` — 言語検証記録、採用形、保留事項、参照資料

## 収録章

1. 私はする — 一人称 `-ēn`、現在、強弱、主語省略
2. 私はした — 過去クラス、過去形と副動詞のペア
3. 私はする・しない — 未来、非未来否定、未来否定、存在否定
4. 私はしたい・できる — 不定詞、`-ṇum`、`muḍiyum`、`nāṉ / eṉakku / eṉṉāle`
5. あなたはする — 丁寧二人称 `nīṅka + -īṅka`、現在・過去・未来
6. あなたはしますか — 極性疑問 `-ā`、`-īṅkaḷ-ā`、肯定・否定応答
7. いつ・どこ・何を — 疑問詞の元位置、疑問詞＋格、`-ā` との区別
8. 質問して、答える — 丁寧二人称の問いから一人称回答への三時制往復
9. 彼は／彼女はする — 男性 `-āṉ`、女性 `-ā`、親称の使用域
10. あの方はする — 男性敬称 `-āru`、女性敬称 `-āṅka`、社会距離
11. 彼らはする — 人物複数 `-āṅka`、`avaṅka` の敬称／複数重複
12. それは起こる — 中性三時制、未来 `-um`、`pōccu / irukku`
13. している・してしまった — 進行 `V-kkiṭṭu iru`、現在関連の完了 `V-iru`、完遂 `V-(v)iḍu`
14. して、行く — 連用形による動詞連鎖、不定詞による目的、有限語尾の終端化
15. してください — 丁寧命令、否定命令、許可、一般禁止
16. 君／私たちはする — 親称 `nī + -ē`、包括 `nāma`、除外 `nāṅka`、一人称複数 `-ōm`、勧誘 `-lām`
17. 〜って言った・思った — 引用・埋込み `-nnu`、意図 `-lām-nnu`、伝聞 `-ām`
18. 〜なら・〜でも・〜ので — 条件 `-ā`、譲歩 `-ālum`、理由、現在／過去連体形、前置型関係節
19. 空港で到着する — 書類、荷物、入国審査、出口
20. ホテルに泊まる — 予約、設備、不具合、依頼
21. 鉄道で移動する — 行先、ホーム、時刻、座席、乗換
22. バス・オートに乗る — 住所、運賃、メーター、降車
23. 道を尋ねて歩く — 左右、位置、目印、徒歩可否
24. 食堂で注文する — 菜食、味、追加、持ち帰り、会計
25. 店と市場で買う — 価格、量、色、サイズ、支払い
26. 寺院を訪ねる — 時刻、履物、撮影規則、祭礼
27. 体調を伝える — 症状、期間、薬局、医師、緊急時
28. 雑談して旅を閉じる — 聞き返し、自己紹介、映画・音楽
29. ある・いる・持っている — 存在、所在、状態、二つの所有、名詞文
30. 物・相手・行先を標識する — 対格 `-e`、与格 `-kku`、所有 `-ōḍa`
31. 場所・起点・一緒・手段 — `-le / -kiṭṭa / -lērundu / -ōḍa / -āle`
32. 複数と名詞の裏の姿 — 口語複数 `-ṅka`、人数 `pēr`、斜格語幹
33. これ・それ・あれを組み直す — 近称 `i-`、遠称 `a-`、疑問 `e-`
34. 数・量・時間を束ねる — 数詞、人数、時刻、所要時間、概数、序数
35. 比べる・いちばんを言う — 比較 `-e viḍa`、最上級範囲 `-le-yē`
36. も・だけ・こそ・何か — `-um / maṭṭum / tāṉ / -ē / -ō / -āvadu`
37. 文を人・もの・出来事にする — 否定連体 `-āda`、名詞化 `-adu / -avaṅka`
38. させる・自分で・される — 使役 `V-a vai`、自己利益 `-kko`、口語の非人称

教材を改稿するときは、画面HTMLへ例文を直書きせず、基礎編は `curriculum.js`、旅行科は `travel-curriculum.js`、文法補完科は `grammar-curriculum.js` を更新してください。書記形と口語音の差、カナ、音声用読み、本文内注記は `transcription-revision.js` へ集約します。同じデータから本文、音声キュー、活用表／語彙帳／文法型、練習、印刷版が生成されます。
