# タミル語 文法の庭｜第1・2課 試作版

スマートフォン向けの、標準的な口語タミル語を中心とした基礎文法アプリです。

## ファイル構成

すべて同じ階層へ置きます。

- `index.html`
- `styles.css`
- `app.js`
- `lesson01.js`
- `lesson02.js`
- `LANGUAGE_CHECK.md`
- `APPROVED_RULES.md`

## 開き方

1. ZIPを展開します。
2. 全ファイルを同じフォルダに置いたまま、`index.html` を開きます。
3. Android版Chromeなど、Web Speech APIに対応するブラウザを推奨します。

端末にタミル語音声が入っていない場合、ブラウザが代替音声を選ぶか、再生できないことがあります。

## 音声操作

- タミル文字の行、または右端の小さな再生印をタップ：個別再生
- 下部の設定：速度と繰返し回数を全項目へ共通適用
- `課内を再生`：現在開いている課の、画面上で見えているタミル語項目を順番に再生
- `■`：停止

例文だけでなく、短い句、代名詞、格変化した完成語にも音声を付けています。
語尾だけを単独再生せず、自然な単語として成立する形を聞く設計です。

## デザイン

チェンナイの海、日差し、マンゴー、ブーゲンビリア、コーラムを抽象化しています。
写真や大きな装飾は使わず、明るい南国感と読みやすさを両立させています。
