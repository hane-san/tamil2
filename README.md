# タミル語 文法の庭｜全20課・標準口語改訂版

スマートフォンを中心に、タブレットでも読みやすい一列型のタミル語基礎文法アプリです。例文を分解しながら、日常、旅行、映画会話に必要な文法を学びます。

## 言語層

教材は三層で表示します。

1. **主表示：広く通じる標準口語**
2. **補助：チェンナイで特に聞きやすい形**
3. **補助：字幕・看板・文章で見る書記形**

例文カードは原則として、次の順です。

1. 標準口語のタミル文字
2. 規格準拠カタカナ
3. 構造が見えるローマ字
4. 自然な日本語
5. 必要な場合だけチェンナイ形・書記形

詳細は `REGISTER_AND_KANA_RULES.md` を参照してください。

## 収録範囲

2. 代名詞と中心となる格

9. 一人称の否定

15. 命令・依頼・勧誘・疑問
16. 名詞句を詳しくする

### 改訂レイヤー

元の20課を別々に手直しして表記を再び揺らさないよう、横断改訂を二つの共通ファイルへ集約しています。

- `lesson-revision.js`：標準口語化、カタカナ再生成、各課の「この課の芯」の追加
- `lesson-ui-revision.js`：カタカナをタミル文字の直下へ移動し、チェンナイ形・書記形を補助表示

## 開き方

1. 全ファイルを同じフォルダへ置きます。
2. `index.html` をSafari、Chrome、Edgeなどで開きます。
3. GitHub Pagesでも追加のビルド処理は不要です。
4. iPhoneでは共有メニューから「ホーム画面に追加」すると、専用アイコン付きで起動できます。

## 音声

音声設定は課選択タブの直下にあり、本文をスクロールすると画面外へ流れます。

- タミル文字をタップ：個別再生
- 速度：ゆっくり／自然／速め
- 回数：1回／2回
- `課内を再生`：表示中のタミル語を順番に再生
- `■`：停止

TTSへ渡すのは標準口語の自然なタミル文字だけです。日本語、ローマ字、カタカナ、解説は読み上げません。

## カタカナ

カタカナは音読開始と想起の補助です。情報保存はタミル文字、構造ローマ字、音声が担います。

- 同じ入力には同じカタカナを返す
- ローマ字ではなくタミル文字と音環境から生成する
- 長母音、重子音、鼻音後の有声化、母音間の有声化を一貫して反映する
- `வ` はヴァ行、`ழ` はラ行、語頭 `எ / ஏ` はイェ／イェー

## 検証状態

全20課へ横断改訂を適用していますが、母語話者による最終承認前です。次を優先して確認します。

1. 標準口語としての自然さ
2. チェンナイ補助形の地域差
3. 書記形から口語形への変換で意味・敬意が変わっていないこと
4. 全カタカナとTamil TTSの照合
5. iPhone・iPadでの表示、課タブ、長文カード、連続再生
6. 20ページ印刷教材との例文・課構成の対応
