# タミル語 文法の庭｜全20課 試作完成版

スマートフォンを中心に、タブレットでも読みやすい一列型の基礎文法アプリです。標準的な口語タミル語への入口を、例文の分解、音声、理由説明、チェンナイ旅行と映画会話への接続から学びます。

## 現在の収録範囲

- 第1課　名詞の形
- 第2課　代名詞と中心となる格
- 第3課　所有と結びつきを表す格
- 第4課　場所と移動を表す格
- 第5課　後置詞と格の組合せ
- 第6課　名詞だけで作る文
- 第7課　動詞の設計図
- 第8課　一人称の現在・過去・未来
- 第9課　一人称の否定
- 第10課　不定詞
- 第11課　副動詞と動作の連結
- 第12課　動作の進み方を表す補助動詞
- 第13課　可能・希望・義務・意志
- 第14課　すべての人称へ広げる
- 第15課　命令・依頼・勧誘・疑問
- 第16課　名詞句を詳しくする
- 第17課　名詞を修飾する動詞
- 第18課　文を名詞として扱う
- 第19課　引用と発言
- 第20課　条件・理由・目的・時間

全20課の教材データは揃いました。ただし、現段階は「言語・実機検証前の試作完成版」です。語彙表、文法索引、横断検索、復習再生リストは、全課監査と第1・2課の承認済み見本化の後に追加します。

## 開き方

1. 全ファイルを同じフォルダへ置きます。
2. `index.html` をSafari、Chrome、Edgeなどで開きます。
3. GitHub Pagesを使う場合も、追加のビルド処理は不要です。

Web Speech APIの実装と端末音声は、ブラウザ・OSによって異なります。タミル語音声が確認できない場合も、教材本文と問題は停止しません。

## 音声操作

- タミル文字の行をタップ：そのタミル語だけを再生
- 下部の設定：速度と繰返し回数を共通適用
- `課内を再生`：現在の課にある、展開中のタミル語項目を順番に再生
- `■`：停止

TTSへ渡すのは `tts` に保存した自然なタミル文字列だけです。日本語、ローマ字、カタカナ、文法解説、ボタン名は読み上げません。

## 第19・20課で追加した考え方

### 第19課

- `என்று` が前の文・語・音を引用内容としてまとめ、後ろの発言・思考・質問動詞へ渡す
- `வருவேன் என்று சொன்னார்`：話した人の一人称を保つ直接引用
- `வருவார் என்று நினைக்கிறேன்`：三人称の内容を思考へ渡す
- `வந்தார் என்று கேள்விப்பட்டேன்`：伝聞
- `எங்கே போகிறாய் என்று கேட்டார்`：疑問詞質問の引用
- `வருவாயா என்று கேட்டார்`：はい・いいえ疑問の引用
- `வா என்று சொன்னார் / வரச் சொன்னார்`：言葉の直接引用と、指示された行為の違い
- `என்பது` は内容を名詞化し、`என்று` は内容を発言・思考動詞へ渡すという入口の対比
- 映画口語の `-ன்னு / -னு` を `என்று` へ戻す

### 第20課

- 最後の有限動詞・述語を見つけ、左側を節末の標識で切る
- `-ஆல்`：もし〜なら／〜したら
- `இல்லையென்றால் / வரவில்லை என்றால் / வராவிட்டால்`：否定条件
- `-ஆலும்`：〜ても／たとえ〜でも
- `-தால்`：〜なので／〜したため
- 不定詞と `-தற்காக`：目的を表す短い形・明示的な形
- `-போது / முன் / பிறகு / வரை`：とき・前・後・まで
- 条件・理由・時間節を重ねた長文を、最後の述語から逆算する
- 映画口語の `வந்தா / இல்லனா / ஏன்னா / வர்றப்போ / வந்தப்புறம்` を主要形へ戻す

## 教材データ

各課は `window.TAMIL_LESSONS` へデータを追加します。主要な例文項目は次のフィールドを使います。

- `tamil`：画面表示するタミル文字
- `tts`：TTSへ渡すタミル文字
- `roman`：文法構造を示すローマ字
- `surface`：必要な場合だけ表示する表面発音形
- `katakana`：日本語話者向けの近似
- `japanese`：自然な日本語訳
- `literal`：必要な場合だけ表示する構造寄りの訳・注記
- `morph`：形態素の短い説明
- `context`：自宅、映画、旅行などの場面タグ

## 次の工程

全課制作の次は、機能追加より先に横断監査を行います。

1. 第1・2課を実機・母語話者確認し、承認済み見本へする
2. 変更点を `APPROVED_RULES.md` へ反映する
3. 第3〜20課を同じ基準で横断修正する
4. 語彙の新出数・再登場回数・未習文法を点検する
5. ローマ字・カタカナ・TTS・口語／書記形の一貫性を点検する
6. その後に語彙表、文法索引、横断検索、復習再生リストを追加する

## 検証の優先順位

1. タミル語母語話者による全主要例文の自然さ
2. `என்று / என்பது / -ன்னு` の境界と引用内の人称
3. `கேட்டேன் / கேள்விப்பட்டேன் / சொன்னார் / நினைத்தேன்` の語用
4. `-ஆல் / -ஆலும் / -தால்` と否定条件の形
5. 目的の不定詞と `-தற்காக` の距離
6. `-போது / முன் / பிறகு / வரை` の時間節
7. 全20課のローマ字分解・カタカナ・TTS文字列
8. iPhone・iPadで20課分の課タブ、長文カード、連続再生が扱いやすいか
