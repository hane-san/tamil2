# タミル語 文法の庭｜第1〜10課 増補版

スマートフォンを中心に、タブレットでも読みやすい一列型の基礎文法アプリです。標準的な口語タミル語への入口を、例文の分解、音声、理由説明、チェンナイ旅行と映画会話への接続から学びます。

## 現在の収録範囲

- 第1課　名詞の形
- 第2課　代名詞と中心となる格
- 第3課　所有と結びつきを表す格
- 第4課　場所と移動を表す格
- 第5課　後置詞と格の組合せ
- 第6課　名詞だけで作る文
- 第7課　動詞の設計図
- 第8課　一人称の現在・過去・未来
- 第9課　一人称の否定
- 第10課　不定詞

第11課以降は未制作です。原則として2課ずつ追加し、各セットを検証してから次へ進みます。

## 開き方

1. 全ファイルを同じフォルダへ置きます。
2. `index.html` をSafari、Chrome、Edgeなどで開きます。
3. GitHub Pagesを使う場合も、追加のビルド処理は不要です。

Web Speech APIの実装と端末音声は、ブラウザ・OSによって異なります。タミル語音声が確認できない場合も、教材本文と問題は停止しません。

## 音声操作

- タミル文字の行をタップ：そのタミル語だけを再生
- 下部の設定：速度と繰返し回数を共通適用
- `課内を再生`：現在の課にある、展開中のタミル語項目を順番に再生
- `■`：停止

TTSへ渡すのは `tts` に保存した自然なタミル文字列だけです。日本語、ローマ字、カタカナ、文法解説、ボタン名は読み上げません。

## 第9・10課で追加した考え方

### 第9課

- 肯定の現在・過去・未来を、そのまま一対一で否定へ反転できないこと
- `பார்க்க மாட்டேன்`：未来否定・非習慣・拒否・意志的な非実現
- `பார்க்கவில்லை`：文脈により「見なかった」「まだ見ていない」「今は見ていない」
- `பார்க்க முடியாது`：能力・状況として「見ることができない」
- `-வில்லை` 型には人称語尾がなく、主語代名詞や文脈が人物を補うこと
- `மாட்டேன் / மாட்டோம்` には一人称末尾が残ること
- 映画口語の `பாக்கல / போகல` を、構造形 `பார்க்கவில்லை / போகவில்லை` へ戻す入口

### 第10課

- 不定詞を「動詞を次の動詞・述語へ渡す非定形」として導入
- `பார் → பார்க்க`、`போ → போக`、`வா → வர`、`செய் → செய்ய` などの頻出ペア
- 不定詞自身は時制・人称を完成させず、最後の有限動詞が文全体を完成させること
- `படம் பார்க்க போகிறேன்`：目的＋移動
- `பேச முடியும்`：可能
- `வர வேண்டும்`：必要・義務
- `போக மாட்டேன்`：非実現・拒否
- `வரச் சொல்லுங்கள்`：内容・指示への接続
- チェンナイ周辺で `ஹோட்டல்` が食堂・レストランを指すことがある文化補足

## 教材データ

各課は `window.TAMIL_LESSONS` へデータを追加します。主要な例文項目は次のフィールドを使います。

- `tamil`：画面表示するタミル文字
- `tts`：TTSへ渡すタミル文字
- `roman`：文法構造を示すローマ字
- `surface`：必要な場合だけ表示する表面発音形
- `katakana`：日本語話者向けの近似
- `japanese`：自然な日本語訳
- `literal`：必要な場合だけ表示する構造寄りの訳・注記
- `morph`：形態素の短い説明
- `context`：自宅、映画、旅行などの場面タグ

## 制作単位

各セットで次を確認してから、次の2課へ進みます。

1. 既習語彙・既習文法との接続
2. 主要口語形と保守的綴りの距離
3. ローマ字分解とタミル文字の整合
4. TTSでの聞こえ方
5. スマホ・タブレットでの長さと密度
6. 文化補足が文法理解へ本当に役立つか

第3〜10課では共通UIを変更せず、教材データだけで新しい内容を表現できるかを検証しています。

## 検証の優先順位

1. タミル語母語話者による例文の自然さ
2. `-வில்லை` の過去・未完了・現在不成立の読み分け
3. `மாட்டேன்` の未来否定・習慣・拒否の範囲
4. `என்னால் ... முடியாது` の能力者表示
5. `பார் → பார்க்க / வா → வர / செய் → செய்ய` の不定詞表示
6. `வரச் சொல்லுங்கள்` の接合表記と丁寧さ
7. 映画口語 `பாக்கல / பாக்கப் போறேன்` の中立性
8. iPhone SafariのTamil TTSと、iPadで10課分のタブ操作
