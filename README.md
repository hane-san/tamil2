# タミル語 文法の庭｜第1〜12課 増補版

スマートフォンを中心に、タブレットでも読みやすい一列型の基礎文法アプリです。標準的な口語タミル語への入口を、例文の分解、音声、理由説明、チェンナイ旅行と映画会話への接続から学びます。

## 現在の収録範囲

- 第1課　名詞の形
- 第2課　代名詞と中心となる格
- 第3課　所有と結びつきを表す格
- 第4課　場所と移動を表す格
- 第5課　後置詞と格の組合せ
- 第6課　名詞だけで作る文
- 第7課　動詞の設計図
- 第8課　一人称の現在・過去・未来
- 第9課　一人称の否定
- 第10課　不定詞
- 第11課　副動詞と動作の連結
- 第12課　動作の進み方を表す補助動詞

第13課以降は未制作です。原則として2課ずつ追加し、各セットを検証してから次へ進みます。

## 開き方

1. 全ファイルを同じフォルダへ置きます。
2. `index.html` をSafari、Chrome、Edgeなどで開きます。
3. GitHub Pagesを使う場合も、追加のビルド処理は不要です。

Web Speech APIの実装と端末音声は、ブラウザ・OSによって異なります。タミル語音声が確認できない場合も、教材本文と問題は停止しません。

## 音声操作

- タミル文字の行をタップ：そのタミル語だけを再生
- 下部の設定：速度と繰返し回数を共通適用
- `課内を再生`：現在の課にある、展開中のタミル語項目を順番に再生
- `■`：停止

TTSへ渡すのは `tts` に保存した自然なタミル文字列だけです。日本語、ローマ字、カタカナ、文法解説、ボタン名は読み上げません。

## 第11・12課で追加した考え方

### 第11課

- 前の動作を副動詞形にし、最後の有限動詞だけで時制・人称を完成させる
- `வந்து / போய் / செய்து / பார்த்து / வாங்கி / சாப்பிட்டு` などの頻出副動詞形
- `காபி குடித்து வீட்டுக்கு போகிறேன்`：順番
- `போன் செய்து கேட்கிறேன்`：方法・準備
- `பார்த்துவிட்டு போனேன்`：前の動作を完了してから次へ進む
- `பேசிக்கொண்டே நடக்கிறேன்`：同時・継続
- `சாப்பிடாமல் வந்தேன்`：前の動作を行わずに次へ進む
- 映画口語の `செஞ்சு / பாத்து / பாத்துட்டு` を、構造形へ戻す聞き方

### 第12課

- 補助動詞を、前の動作の進み方・完了・準備・試行を示すものとして導入
- `பார்த்துக்கொண்டிருக்கிறேன்`：進行を明示
- `பேசிக்கொண்டே இருக்கிறார்`：継続
- `படித்து முடித்தேன்`：最後までの完了
- `பார்த்துவிட்டேன்`：完了・既済・話し手の態度
- `வாங்கி வைத்தேன்`：結果を後のために保持する
- `கேட்டுப் பார்த்தேன் / சாப்பிட்டுப் பார்த்தேன்`：試行
- 映画口語の `பாத்துக்கிட்டு இருக்கேன் / பாத்துட்டேன் / வாங்கி வெச்சேன்` を基本形へ戻す

## 教材データ

各課は `window.TAMIL_LESSONS` へデータを追加します。主要な例文項目は次のフィールドを使います。

- `tamil`：画面表示するタミル文字
- `tts`：TTSへ渡すタミル文字
- `roman`：文法構造を示すローマ字
- `surface`：必要な場合だけ表示する表面発音形
- `katakana`：日本語話者向けの近似
- `japanese`：自然な日本語訳
- `literal`：必要な場合だけ表示する構造寄りの訳・注記
- `morph`：形態素の短い説明
- `context`：自宅、映画、旅行などの場面タグ

## 制作単位

各セットで次を確認してから、次の2課へ進みます。

1. 既習語彙・既習文法との接続
2. 主要口語形と保守的綴りの距離
3. ローマ字分解とタミル文字の整合
4. TTSでの聞こえ方
5. スマホ・タブレットでの長さと密度
6. 文化補足が文法理解へ本当に役立つか

第3〜12課では共通UIを変更せず、教材データだけで新しい内容を表現できるかを検証しています。

## 次回の制作範囲

次は第13課「可能・希望・義務・意志」と第14課「すべての人称へ広げる」です。第13課では第9〜12課の助動詞構造を整理し、第14課では男性・女性・敬意・人間複数・非人間形まで時制表を広げます。

## 検証の優先順位

1. タミル語母語話者による例文の自然さ
2. `வந்து / போய் / செய்து / பார்த்து / வாங்கி / சாப்பிட்டு` の副動詞表示
3. 副動詞列で通常同じ主語を共有する説明の範囲
4. `-விட்டு / -கொண்டே / -ஆமல்` の意味と口語形
5. `பார்த்துக்கொண்டிருக்கிறேன்` の進行と単純現在の対比
6. `முடி / விடு / வை / பார்` の補助動詞用法
7. `பாத்துட்டேன் / வாங்கி வெச்சேன்` などの映画口語
8. iPhone SafariのTamil TTSと、iPadで12課分のタブ操作
