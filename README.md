# タミル語 文法の庭｜第1〜16課 増補版

スマートフォンを中心に、タブレットでも読みやすい一列型の基礎文法アプリです。標準的な口語タミル語への入口を、例文の分解、音声、理由説明、チェンナイ旅行と映画会話への接続から学びます。

## 現在の収録範囲

- 第1課　名詞の形
- 第2課　代名詞と中心となる格
- 第3課　所有と結びつきを表す格
- 第4課　場所と移動を表す格
- 第5課　後置詞と格の組合せ
- 第6課　名詞だけで作る文
- 第7課　動詞の設計図
- 第8課　一人称の現在・過去・未来
- 第9課　一人称の否定
- 第10課　不定詞
- 第11課　副動詞と動作の連結
- 第12課　動作の進み方を表す補助動詞
- 第13課　可能・希望・義務・意志
- 第14課　すべての人称へ広げる
- 第15課　命令・依頼・勧誘・疑問
- 第16課　名詞句を詳しくする

第17課以降は未制作です。原則として2課ずつ追加し、各セットを検証してから次へ進みます。

## 開き方

1. 全ファイルを同じフォルダへ置きます。
2. `index.html` をSafari、Chrome、Edgeなどで開きます。
3. GitHub Pagesを使う場合も、追加のビルド処理は不要です。

Web Speech APIの実装と端末音声は、ブラウザ・OSによって異なります。タミル語音声が確認できない場合も、教材本文と問題は停止しません。

## 音声操作

- タミル文字の行をタップ：そのタミル語だけを再生
- 下部の設定：速度と繰返し回数を共通適用
- `課内を再生`：現在の課にある、展開中のタミル語項目を順番に再生
- `■`：停止

TTSへ渡すのは `tts` に保存した自然なタミル文字列だけです。日本語、ローマ字、カタカナ、文法解説、ボタン名は読み上げません。

## 第15・16課で追加した考え方

### 第15課

- 親しい一人への命令と、丁寧・複数への依頼を分ける
- `வா / வாருங்கள்`、`பார் / பாருங்கள்`、`சொல் / சொல்லுங்கள்` の対比
- `கொஞ்சம்` と `முடியுமா?` による依頼の軟化
- `போகாதே / போகாதீர்கள்` の否定命令
- `போக மாட்டேன்` と `போகாதே` の、話し手・聞き手の違い
- `நாம போகலாம் / போகலாமா?` の勧誘・提案
- `நான் உள்ளே போகலாமா?` の許可
- 述語へ `-ஆ` を付ける、はい・いいえ疑問
- `யார் / என்ன / எங்கே / எப்போது / ஏன் / எப்படி / எவ்வளவு` の疑問詞
- 映画口語の `வாங்க / பாருங்க / சொல்லுங்க / வரீங்களா` を完全形へ戻す

### 第16課

- 指示・所有・数・性質を名詞の前へ積み、最後の名詞へまとめる
- `அந்த இரண்டு புதிய தமிழ் படங்கள்` の語順
- 複数・格語尾は名詞句最後の中心名詞へ付く
- `நல்ல / பெரிய / சின்ன / பழைய / புதிய` は名詞の性・数・格で変化しない
- `அழகான / நீளமான / சூடான / மலிவான` の `-ஆன` 形容表現
- `ஒரு / இரண்டு / சில / நிறைய / கொஞ்சம்` の数・数量
- `முதல் / இரண்டாவது` の順序
- `வேகமான` と `வேகமாக` の、形容詞・副詞の対比
- 対格形＋`விட` による比較
- 集合＋`மிகவும்` による最上級的な表現
- 会話の `ரொம்ப நல்ல படம்` と、中心名詞の省略

## 教材データ

各課は `window.TAMIL_LESSONS` へデータを追加します。主要な例文項目は次のフィールドを使います。

- `tamil`：画面表示するタミル文字
- `tts`：TTSへ渡すタミル文字
- `roman`：文法構造を示すローマ字
- `surface`：必要な場合だけ表示する表面発音形
- `katakana`：日本語話者向けの近似
- `japanese`：自然な日本語訳
- `literal`：必要な場合だけ表示する構造寄りの訳・注記
- `morph`：形態素の短い説明
- `context`：自宅、映画、旅行などの場面タグ

## 検証の優先順位

1. タミル語母語話者による例文の自然さ
2. `வா / வாருங்கள் / வாங்க` の距離と多義性
3. `-ஆதே / -ஆதீர்கள்` の否定命令
4. `-லாம் / -லாமா` の勧誘・許可・可能性
5. はい・いいえ疑問の `-ஆ` と口語縮約
6. 名詞句の修飾語順と、格語尾の位置
7. 数詞後の複数語尾の標準語・口語差
8. `-ஆன / -ஆக`、`விட`、`மிகவும்` の比較表現
9. iPhone SafariのTamil TTSと、iPadで16課分のタブ操作
