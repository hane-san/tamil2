# タミル語 文法の庭｜第1〜8課 増補版

スマートフォンを中心に、タブレットでも読みやすい一列型の基礎文法アプリです。標準的な口語タミル語への入口を、例文の分解、音声、理由説明、チェンナイ旅行と映画会話への接続から学びます。

## 現在の収録範囲

- 第1課　名詞の形
- 第2課　代名詞と中心となる格
- 第3課　所有と結びつきを表す格
- 第4課　場所と移動を表す格
- 第5課　後置詞と格の組合せ
- 第6課　名詞だけで作る文
- 第7課　動詞の設計図
- 第8課　一人称の現在・過去・未来

第9課以降は未制作です。原則として2課ずつ追加し、各セットを検証してから次へ進みます。

## 開き方

1. 全ファイルを同じフォルダへ置きます。
2. `index.html` をSafari、Chrome、Edgeなどで開きます。
3. GitHub Pagesを使う場合も、追加のビルド処理は不要です。

Web Speech APIの実装と端末音声は、ブラウザ・OSによって異なります。タミル語音声が確認できない場合も、教材本文と問題は停止しません。

## 音声操作

- タミル文字の行をタップ：そのタミル語だけを再生
- 下部の設定：速度と繰返し回数を共通適用
- `課内を再生`：現在の課にある、展開中のタミル語項目を順番に再生
- `■`：停止

TTSへ渡すのは `tts` に保存した自然なタミル文字列だけです。日本語、ローマ字、カタカナ、文法解説、ボタン名は読み上げません。

## 第7・8課で追加した考え方

### 第7課

- 動詞を `動詞の核―時制を作る部分―人称語尾` の順で分解する
- `பார்க்கிறேன் pār-kkiṟ-ēṉ` の末尾 `-ēṉ` が一人称単数を示す
- 動詞ごとに過去・未来の作り方が異なるため、活用上の動詞クラスが必要になる
- `செய் / பார் / படி / சாப்பிடு` を三時制で比較する
- 自動詞と他動詞を、目的語へ向かう矢印の有無として導入する
- `வா / போ / பார் / படி / சாப்பிடு / கேள்` の短い命令形から動詞の核を見る
- 映画口語の `பாக்கறேன் / பாக்குறேன் / போறேன்` は文化カードで補足し、主要分析には混在させない

### 第8課

- `பார்க்கிறேன் / பார்த்தேன் / பார்ப்பேன்` を現在・過去・未来の一列で比較する
- 一人称単数 `-ēṉ` と一人称複数 `-ōm` は、時制が変わっても末尾に残る
- 単純現在が、文脈により進行中の動作と習慣の両方を表す
- `போ / சாப்பிடு / படி / செய்` の頻出動詞を三時制で束ねる
- 一人称未来が、予測だけでなく意志・予定・約束を表す
- `நாம ... பார்ப்போம்` のような一人称複数未来が勧誘として読まれる入口を示す
- 主語代名詞が省かれても、動詞末尾から人物を復元できることを確認する

## 教材データ

各課は `window.TAMIL_LESSONS` へデータを追加します。主要な例文項目は次のフィールドを使います。

- `tamil`：画面表示するタミル文字
- `tts`：TTSへ渡すタミル文字
- `roman`：文法構造を示すローマ字
- `surface`：必要な場合だけ表示する表面発音形
- `katakana`：日本語話者向けの近似
- `japanese`：自然な日本語訳
- `literal`：必要な場合だけ表示する構造寄りの訳・注記
- `morph`：形態素の短い説明
- `context`：自宅、映画、旅行などの場面タグ

## 制作単位

各セットで次を確認してから、次の2課へ進みます。

1. 既習語彙・既習文法との接続
2. 主要口語形と保守的綴りの距離
3. ローマ字分解とタミル文字の整合
4. TTSでの聞こえ方
5. スマホ・タブレットでの長さと密度
6. 文化補足が文法理解へ本当に役立つか

第3〜8課では共通UIを変更せず、教材データだけで新しい内容を表現できるかを検証しています。

## 検証の優先順位

1. タミル語母語話者による例文の自然さ
2. `பார் / படி / சாப்பிடு / செய் / போ` の三時制分解
3. `-kiṟ-` を現在標識として示す説明と、口語縮約との距離
4. `போனேன்` の `pō-ṉ-ēṉ`、`சாப்பிட்டேன்` の `cāppiṭṭ-ēṉ` などのハイフン判断
5. 単純現在の習慣・進行中の読み分け
6. 一人称未来の意志・予定と、`-ōm` の勧誘用法
7. iPhone SafariでのTTS
8. iPad縦・横画面で8つの課タブが扱いやすいか
