# タミル語 文法の庭｜第1〜6課 増補版

スマートフォンを中心に、タブレットでも読みやすい一列型の基礎文法アプリです。標準的な口語タミル語への入口を、例文の分解、音声、理由説明、チェンナイ旅行と映画会話への接続から学びます。

## 現在の収録範囲

- 第1課　名詞の形
- 第2課　代名詞と中心となる格
- 第3課　所有と結びつきを表す格
- 第4課　場所と移動を表す格
- 第5課　後置詞と格の組合せ
- 第6課　名詞だけで作る文

第7課以降は未制作です。以後も原則として2課ずつ追加し、各セットを検証してから次へ進みます。

## 開き方

1. 全ファイルを同じフォルダへ置きます。
2. `index.html` をSafari、Chrome、Edgeなどで開きます。
3. GitHub Pagesを使う場合も、追加のビルド処理は不要です。

Web Speech APIの実装と端末音声は、ブラウザ・OSによって異なります。タミル語音声が確認できない場合も、教材本文と問題は停止しません。

## 音声操作

- タミル文字の行をタップ：そのタミル語だけを再生
- 下部の設定：速度と繰返し回数を共通適用
- `課内を再生`：現在の課にある、展開中のタミル語項目を順番に再生
- `■`：停止

TTSへ渡すのは `tts` に保存した自然なタミル文字列だけです。日本語、ローマ字、カタカナ、文法解説、ボタン名は読み上げません。

## 第5・6課で追加した考え方

### 第5課

- 後置詞は、名詞の後ろに置くだけでなく、前の名詞へ特定の格形を要求する
- `படத்தைப் பற்றி`：対格＋「〜について」
- `தியேட்டர் வரை`：基本形＋「〜まで」
- `நண்பருக்காக`：与格を土台にした「〜のために」
- `காபிக்குப் பதிலாக`：与格＋「〜の代わりに」
- `அவரைத் தவிர`：対格＋「〜を除いて」
- 口語では `படம் பற்றி` のように、`பற்றி` の前の対格が表面に出ない場合がある

### 第6課

- 現在の名詞文では、英語の be 動詞に相当する語を置かずに `இது என் வீடு` と言える
- 同一・分類と、`இருக்கிறது` による存在・所在を分ける
- `இல்லை` が不在・非所有・名詞文の否定を担う
- `என்னிடம் ... இருக்கிறது` による所有
- `எனக்கு ... பிடிக்கும் / வேண்டும் / புரியும்` による与格経験者
- `இது நல்ல படம்` と `இந்த படம் நல்லது` を、名詞修飾と述語に分ける

## 教材データ

各課は `window.TAMIL_LESSONS` へデータを追加します。主要な例文項目は次のフィールドを使います。

- `tamil`：画面表示するタミル文字
- `tts`：TTSへ渡すタミル文字
- `roman`：文法構造を示すローマ字
- `surface`：必要な場合だけ表示する表面発音形
- `katakana`：日本語話者向けの近似
- `japanese`：自然な日本語訳
- `literal`：必要な場合だけ表示する構造寄りの訳・注記
- `morph`：形態素の短い説明
- `context`：自宅、映画、旅行などの場面タグ

## 2課ずつ進める理由

各セットで次を確認してから、次の2課へ進みます。

1. 既習語彙・既習文法との接続
2. 主要口語形と保守的綴りの距離
3. ローマ字分解とタミル文字の整合
4. TTSでの聞こえ方
5. スマホ・タブレットでの長さと密度
6. 文化補足が文法理解へ本当に役立つか

第3〜6課では共通UIを変更せず、教材データだけで新しい内容を表現できるかを検証しています。

## 検証の優先順位

1. タミル語母語話者による例文の自然さ
2. `பற்றி` の前の対格の有無と口語度
3. `-க்காக / பதிலாக / தவிர` の格支配と接合表記
4. `இது என் வீடு இல்லை` の口語自然性
5. `என்னிடம்` と `எனக்கு` の所有・経験者としての分布
6. `வேண்டும் / வேணும்`、`இருக்கிறது / இருக்கு` の表示方針
7. iPhone SafariでのTTS
8. iPad縦・横画面で6つの課タブが扱いやすいか
