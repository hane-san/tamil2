# タミル語 文法の庭｜第1〜18課 増補版

スマートフォンを中心に、タブレットでも読みやすい一列型の基礎文法アプリです。標準的な口語タミル語への入口を、例文の分解、音声、理由説明、チェンナイ旅行と映画会話への接続から学びます。

## 現在の収録範囲

- 第1課　名詞の形
- 第2課　代名詞と中心となる格
- 第3課　所有と結びつきを表す格
- 第4課　場所と移動を表す格
- 第5課　後置詞と格の組合せ
- 第6課　名詞だけで作る文
- 第7課　動詞の設計図
- 第8課　一人称の現在・過去・未来
- 第9課　一人称の否定
- 第10課　不定詞
- 第11課　副動詞と動作の連結
- 第12課　動作の進み方を表す補助動詞
- 第13課　可能・希望・義務・意志
- 第14課　すべての人称へ広げる
- 第15課　命令・依頼・勧誘・疑問
- 第16課　名詞句を詳しくする
- 第17課　名詞を修飾する動詞
- 第18課　文を名詞として扱う

第19課以降は未制作です。原則として2課ずつ追加し、各セットを検証してから次へ進みます。

## 開き方

1. 全ファイルを同じフォルダへ置きます。
2. `index.html` をSafari、Chrome、Edgeなどで開きます。
3. GitHub Pagesを使う場合も、追加のビルド処理は不要です。

Web Speech APIの実装と端末音声は、ブラウザ・OSによって異なります。タミル語音声が確認できない場合も、教材本文と問題は停止しません。

## 音声操作

- タミル文字の行をタップ：そのタミル語だけを再生
- 下部の設定：速度と繰返し回数を共通適用
- `課内を再生`：現在の課にある、展開中のタミル語項目を順番に再生
- `■`：停止

TTSへ渡すのは `tts` に保存した自然なタミル文字列だけです。日本語、ローマ字、カタカナ、文法解説、ボタン名は読み上げません。

## 第17・18課で追加した考え方

### 第17課

- 動詞を名詞の前へ置き、英語の関係代名詞なしで名詞を修飾する
- `பார்க்கிற படம்`：現在・進行に関わる映画
- `பார்த்த படம்`：見た映画
- `பார்க்கும் படம்`：これから見る／普段見る映画
- `பார்க்காத படம்`：見ない／まだ見ていない映画
- 連体形は人称語尾を持たず、主語は前の語や文脈から判断する
- `நேற்று வந்த நண்பர்` と `நான் பார்த்த படம்` を比べ、中心名詞が連体節内で果たす役割を読む
- 連体節を含む名詞句でも、複数・格語尾は最後の中心名詞へ付く
- `நான் பார்த்த படத்தை / நேற்று வந்த நண்பருக்கு / நான் போகும் ஹோட்டலில்`
- 映画口語の `பாக்கற படம் / வர்ற ஆள் / பாத்த படம்` を基本形へ戻す

### 第18課

- 動作・出来事・文全体を主語や目的語として扱う
- `பார்ப்பது / பார்த்தது / பார்க்காதது`：見ること・見たこと・見ないこと
- `நான் பார்த்த படம்` と `நான் பார்த்தது` の、中心名詞あり／名詞化の違い
- `நான் படம் பார்த்தது`：私が映画を見たという出来事全体
- `தமிழ் படிப்பது சுவாரஸ்யம்`：名詞化された文が主語
- `எனக்கு தமிழ் படம் பார்ப்பது பிடிக்கும்`：経験の対象
- `அவர் சொன்னதை புரிந்துகொண்டேன்`：名詞化＋対格
- `படம் பார்ப்பதற்கு நேரம் இல்லை`：名詞化＋与格
- `என்பது` による「〜ということ・事実」
- 書記的な `-தல்` 名詞化は補足に留め、口語中心では `-வது / -து` を使う
- 映画口語の `பாக்கறது / வர்றது / பாத்தது / பாக்காதது` を基本形へ戻す

## 教材データ

各課は `window.TAMIL_LESSONS` へデータを追加します。主要な例文項目は次のフィールドを使います。

- `tamil`：画面表示するタミル文字
- `tts`：TTSへ渡すタミル文字
- `roman`：文法構造を示すローマ字
- `surface`：必要な場合だけ表示する表面発音形
- `katakana`：日本語話者向けの近似
- `japanese`：自然な日本語訳
- `literal`：必要な場合だけ表示する構造寄りの訳・注記
- `morph`：形態素の短い説明
- `context`：自宅、映画、旅行などの場面タグ

## 制作単位

各セットで次を確認してから、次の2課へ進みます。

1. 既習語彙・既習文法との接続
2. 主要口語形と保守的綴りの距離

4. TTSでの聞こえ方
5. スマホ・タブレットでの長さと密度
6. 文化補足が文法理解へ本当に役立つか

第3〜18課では共通UIを変更せず、教材データだけで新しい内容を表現できるかを検証しています。

## 検証の優先順位

1. タミル語母語話者による例文の自然さ
2. `-கிற / -த்த / -உம் / -ஆத` の連体形表示
3. 連体節の中心名詞が主語・目的語・場所・相手となる例の自然さ
4. 長い名詞句への対格・与格・処格の付け方
5. `பார்ப்பது / பார்த்தது / பார்க்காதது` の名詞化表示
6. 「もの」と「出来事・事実」の文脈的な読み分け
7. `-தை / -தற்கு` の格付き名詞化
8. `என்பது` と次課の `என்று` の境界
9. 映画口語の `-ற / -றது` と、iPadで18課分のタブ操作
