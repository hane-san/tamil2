# タミル語 文法の庭｜第1〜14課 増補版

スマートフォンを中心に、タブレットでも読みやすい一列型の基礎文法アプリです。標準的な口語タミル語への入口を、例文の分解、音声、理由説明、チェンナイ旅行と映画会話への接続から学びます。

## 現在の収録範囲

- 第1課　名詞の形
- 第2課　代名詞と中心となる格
- 第3課　所有と結びつきを表す格
- 第4課　場所と移動を表す格
- 第5課　後置詞と格の組合せ
- 第6課　名詞だけで作る文
- 第7課　動詞の設計図
- 第8課　一人称の現在・過去・未来
- 第9課　一人称の否定
- 第10課　不定詞
- 第11課　副動詞と動作の連結
- 第12課　動作の進み方を表す補助動詞
- 第13課　可能・希望・義務・意志
- 第14課　すべての人称へ広げる

第15課以降は未制作です。原則として2課ずつ追加し、各セットを検証してから次へ進みます。

## 開き方

1. 全ファイルを同じフォルダへ置きます。
2. `index.html` をSafari、Chrome、Edgeなどで開きます。
3. GitHub Pagesを使う場合も、追加のビルド処理は不要です。

Web Speech APIの実装と端末音声は、ブラウザ・OSによって異なります。タミル語音声が確認できない場合も、教材本文と問題は停止しません。

## 音声操作

- タミル文字の行をタップ：そのタミル語だけを再生
- 下部の設定：速度と繰返し回数を共通適用
- `課内を再生`：現在の課にある、展開中のタミル語項目を順番に再生
- `■`：停止

TTSへ渡すのは `tts` に保存した自然なタミル文字列だけです。日本語、ローマ字、カタカナ、文法解説、ボタン名は読み上げません。

## 第13・14課で追加した考え方

### 第13課

- 不定詞の後ろを変え、可能・希望・必要・禁止・意志を作る
- `பேச முடியும்`：能力・条件として可能
- `போக ஆசை இருக்கிறது`：個人的な願い
- `வர வேண்டும்`：希望・必要・義務を文脈から判定
- `பார்க்கவே வேண்டும்`：必要を強調
- `பயன்படுத்தக் கூடாது`：能力ではなく、規則・許可としての禁止
- `வாங்கப் போகிறேன்`：具体化した予定・近い未来
- 映画口語の `வேணும் / -ணும் / போறேன்` を構造形へ戻す

### 第14課

- 人称語尾を「話し手・聞き手」「人間三人称」「非人間」の三群に分ける
- `-ēṉ / -ōm / -āy / -īrkaḷ`：一・二人称
- `-āṉ / -āḷ / -ār / -ārkaḷ`：男性・女性・敬意・人間複数
- 同じ人称語尾が現在・過去・未来の時制幹へ付くことを確認
- `அவர் ... -ஆர்` を性別に依存しない敬意形として扱う
- 非人間単数の `வருகிறது / வந்தது / வரும்`
- 非人間複数の `வருகின்றன / வந்தன / வரும்` という不対称
- 主語代名詞が省かれても動詞末尾から人物を復元する
- 映画口語の `பாக்கற / பாக்கறீங்க / பாக்கறான் / பாக்கறா / பாக்கறாங்க / வருது` を基本形へ戻す

## 教材データ

各課は `window.TAMIL_LESSONS` へデータを追加します。主要な例文項目は次のフィールドを使います。

- `tamil`：画面表示するタミル文字
- `tts`：TTSへ渡すタミル文字
- `roman`：文法構造を示すローマ字
- `surface`：必要な場合だけ表示する表面発音形
- `katakana`：日本語話者向けの近似
- `japanese`：自然な日本語訳
- `literal`：必要な場合だけ表示する構造寄りの訳・注記
- `morph`：形態素の短い説明
- `context`：自宅、映画、旅行などの場面タグ

## 制作単位

各セットで次を確認してから、次の2課へ進みます。

1. 既習語彙・既習文法との接続
2. 主要口語形と保守的綴りの距離
3. ローマ字分解とタミル文字の整合
4. TTSでの聞こえ方
5. スマホ・タブレットでの長さと密度
6. 文化補足が文法理解へ本当に役立つか

第3〜14課では共通UIを変更せず、教材データだけで新しい内容を表現できるかを検証しています。

## 次回の制作範囲

次は第15課「命令・依頼・勧誘・疑問」と第16課「名詞句を詳しくする」です。命令の丁寧さ、否定命令、はい・いいえ疑問、疑問詞を整理した後、形容表現・数・比較・語順へ進みます。

## 検証の優先順位

1. タミル語母語話者による例文の自然さ
2. `முடியும் / முடியாது` と能力者 `என்னால்` の分布
3. `ஆசை இருக்கிறது / வேண்டும் / விரும்புகிறேன்` の希望表現の距離
4. `வேண்டும் / -வே வேண்டும் / கூடாது` の強さ
5. `-ப் போகிறேன்` の移動目的・近接未来・意志の範囲
6. `-āy / -īrkaḷ / -āṉ / -āḷ / -ār / -ārkaḷ` の三時制対応
7. 非人間複数の `வருகின்றன / வந்தன / வரும்`
8. 映画口語の人称語尾と、iPadで14課分のタブ操作
