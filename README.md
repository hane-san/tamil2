# タミル語 文法の庭｜第1〜4課 増補版

スマートフォンを中心に、タブレットでも読みやすい一列型の基礎文法アプリです。標準的な口語タミル語への入口を、例文の分解、音声、理由説明、チェンナイ旅行と映画会話への接続から学びます。

## 現在の収録範囲

- 第1課　名詞の形
- 第2課　代名詞と中心となる格
- 第3課　所有と結びつきを表す格
- 第4課　場所と移動を表す格

第5課以降は未制作です。以後も原則として2課ずつ追加し、各セットを検証してから次へ進みます。

すべて同じ階層へ置きます。

## 開き方

1. 全ファイルを同じフォルダへ置きます。
2. `index.html` をSafari、Chrome、Edgeなどで開きます。
3. GitHub Pagesを使う場合も、追加のビルド処理は不要です。

Web Speech APIの実装と端末音声は、ブラウザ・OSによって異なります。タミル語音声が確認できない場合も、教材本文と問題は停止しません。

## 音声操作

- タミル文字の行をタップ：そのタミル語だけを再生
- 下部の設定：速度と繰返し回数を共通適用
- `課内を再生`：現在の課にある、展開中のタミル語項目を順番に再生
- `■`：停止

TTSへ渡すのは `tts` に保存した自然なタミル文字列だけです。日本語、ローマ字、カタカナ、文法解説、ボタン名は読み上げません。

## 今回追加した考え方

### 第3課

- `என் / உன்` と口語の `-ஓட -ōṭa` による所有・関係
- 同じ `-ஓட` が、後続する名詞や動詞との関係により「〜の」「〜と」を表すこと
- 共同の相手と、`-ஆல் -āl` で示す道具・手段の区別
- 日本語の「で」から機械的に格を選ばず、場所・乗り物・道具を分けること

### 第4課

- `-il`（位置）、`-ukku`（到達点）、`-il-iruntu`（起点）の三方向
- 物・場所の `-il` と、人の領域を示す `iṭam` の基本対照
- `வீட்டுக்குள்`、`முன்னால்`、`பின்னால்`、`பக்கத்தில்` などの空間表現
- `வீட்டுல`、`என்கிட்ட` のような映画口語を、基本構造へ戻して聞く入口

## 教材データ

各課は `window.TAMIL_LESSONS` へデータを追加します。主要な例文項目は次のフィールドを使います。

- `tamil`：画面表示するタミル文字
- `tts`：TTSへ渡すタミル文字
- `roman`：文法構造を示すローマ字
- `surface`：必要な場合だけ表示する表面発音形
- `katakana`：日本語話者向けの近似
- `japanese`：自然な日本語訳
- `literal`：必要な場合だけ表示する構造寄りの訳・注記
- `morph`：形態素の短い説明
- `context`：自宅、映画、旅行などの場面タグ

## 2課ずつ進める理由

各セットで次を確認してから、次の2課へ進みます。

1. 既習語彙・既習文法との接続
2. 主要口語形と保守的綴りの距離
3. ローマ字分解とタミル文字の整合
4. TTSでの聞こえ方
5. スマホ・タブレットでの長さと密度
6. 文化補足が文法理解へ本当に役立つか

第3・4課では既存UIを変更せず、教材データだけで新しい内容を表現できるかも検証しています。

## 検証の優先順位

1. タミル語母語話者による例文の自然さ
2. `-ஓட` の所有・共同用法と `சேர்ந்து` の併用
3. `-ஆல்` と乗り物の `-il` の対照
4. `-il / iṭam` と口語の `-ல / -கிட்ட` の距離
5. 起点形 `-il-iruntu / iṭam-iruntu` の口語音
6. iPhone SafariでのTTS
7. iPad縦・横画面での課タブとカード密度
